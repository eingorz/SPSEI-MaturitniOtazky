# Metody odeslání formuláře
### GET
Data jsou v URL.
Příklad:
`example.com/index.php?jmeno=Jan`
Výhody:
- jednoduché
- lze sdílet odkaz

Nevýhody:
- omezená délka
- není bezpečné

### POST
Data jsou odeslána v těle HTTP požadavku.
Výhody:
- větší množství dat
- bezpečnější

Použití:
- registrace
- přihlášení

## Globální pole
### $_GET
obsahuje data z metody GET
```php
$jmeno = $_GET["jmeno"];
```
### $_POST
obsahuje data z metody POST
```php
$jmeno = $_POST["jmeno"];
```
### $_FILES
obsahuje informace o nahraných souborech.
## Funkce pro kontrolu dat
### isset()
kontroluje existenci proměnné
```php
isset($_POST["jmeno"])
```
### empty()
kontroluje prázdnou hodnotu
```php
empty($_POST["jmeno"])
```
