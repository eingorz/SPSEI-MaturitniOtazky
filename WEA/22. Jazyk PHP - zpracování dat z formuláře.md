# Metody odeslání formuláře
Formulář na webu může odesílat data dvěma hlavními způsoby: **GET** a **POST**.
## GET
Data jsou odeslána v URL.

Příklad:
`example.com/index.php?jmeno=Jan&vek=25`

Výhody:
- jednoduché použití
- lze sdílet odkaz s předvyplněnými hodnotami
- dobré pro vyhledávání nebo filtrování

Nevýhody:
- omezená délka URL (cca 2000 znaků)
- data jsou viditelná v adrese → není bezpečné
- nevhodné pro citlivé údaje (hesla, osobní data)

Použití:
- vyhledávání
- filtry
- sdílení parametrů přes odkaz

## POST
Data jsou odeslána v těle HTTP požadavku a nejsou vidět v URL.

Výhody:
- Lze odeslat větší množství dat
- Bezpečnější než GET (data nejsou vidět v URL)
- Podporuje upload souborů

Nevýhody:
- Nelze snadno sdílet odkaz s daty
- Data nejsou přímo ukládána do historie prohlížeče

Použití:
- Registrace a přihlášení uživatelů
- Odesílání formulářů s citlivými daty
- Upload souborů

# Globální pole
PHP poskytuje předpřipravená globální(všude přístupné) pole pro práci s odeslanými daty:
### $_GET
obsahuje data z metody GET
```php
$jmeno = $_GET["jmeno"];
$vek = $_GET["vek"];
```
### $_POST
obsahuje data z metody POST
```php
$jmeno = $_POST["jmeno"];
$vek = $_POST["vek"];
```
### $_FILES
obsahuje informace o nahraných souborech.
```php
$_FILES["soubor"]["name"]      // původní název souboru
$_FILES["soubor"]["tmp_name"]  // dočasný soubor na serveru
$_FILES["soubor"]["size"]      // velikost souboru
$_FILES["soubor"]["type"]      // MIME typ
```

## Vestavěné funkce pro kontrolu dat
### isset()
Kontroluje, zda proměnná existuje.
```php
if (isset($_POST["jmeno"])) {
    echo "Proměnná je nastavena";
}
```
### empty()
Kontroluje, zda proměnná nemá hodnotu nebo je prázdná.
```php
if (empty($_POST["jmeno"])) {
    echo "Pole jméno je prázdné";
}
```

#### Kombinace pro bezpečné zpracování dat z formuláře
```php
if (isset($_POST["jmeno"]) && !empty($_POST["jmeno"])) {
    $jmeno = $_POST["jmeno"];
    echo "Zadané jméno: $jmeno";
} else {
    echo "Zadejte jméno";
}
```
