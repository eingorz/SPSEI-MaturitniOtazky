# Skriptování na straně klienta
JavaScript běží v prohlížeči.
Používá se pro:
- validaci formulářů
- animace
- dynamický obsah
## Připojení
```html
<script src="script.js"></script>
```
## Proměnné
```js
let x = 10;  
const y = 20;
```
## DOM
DOM je objektová reprezentace HTML dokumentu.
Výběr prvku:
```js
document.getElementById()  
querySelector()
```
## Události
Například:
- click  
- mouseover  
- submit

Registrace:
```js
addEventListener()
```
## Manipulace
### Manipulace HTML
```js
element.innerHTML
```
### Manipulace CSS:
```js
element.style.color="red";
```
