# Skriptování na straně klienta - JavaScript
JavaScript (JS) je programovací jazyk pro web, který běží v prohlížeči. Umožňuje webovým stránkám být interaktivními a dynamickými. Spolupracuje s HTML a CSS kódem. Umí reagovat na uživatele: kliknutí, psaní, pohyb myší.

Můžeme ho použít i na serveru (Node.js), ale hlavní využití je klient – prohlížeč.

Hlavní vlastnosti JavaScriptu:
- Dynamický jazyk – proměnné mohou měnit typ hodnoty.
- Interpretovaný – nepotřebuje kompilaci, prohlížeč spustí kód hned.
- Událostmi řízený – reaguje na akce uživatele.
- Objektově orientovaný – všechno je objekt (např. DOM elementy).

Použití JavaScriptu:
- Validace formulářů před odesláním
- Animace a efekty
- Dynamický obsah (změna textu, obrázků, seznamů bez reloadu)
- Asynchronní načítání dat (AJAX)

Skriptování na straně klienta se liší od skriptování na straně serveru (např. PHP), kde kód běží na serveru a ne v prohlížeči.

## Připojení JavaScriptu k HTML
Externí soubor (doporučeno):
```html
<script src="script.js"></script>
```
Interní skript (přímo v HTML):
```html
<script>
    alert("Ahoj světe!");
</script>
```
Inline skript (v atributech elementů, méně doporučeno):
```html
<button onclick="alert('Klik!')">Klikni</button>
```

Připojení je nadřazené, nejvyšší je **inline -> interní -> externí**.

## Proměnné
Proměnné slouží k ukládání hodnot.
### Proměnná *let*
Lokální proměnné, které lze přepisovat a jejichž platnost je omezena na blok kde je deklarována, přičemž každou z nich lze volitelně inicializovat hodnotou.
```js
let x = 1;
if (x === 1) {
  let x = 2;
  console.log(x);
  // output: 2
}
console.log(x);
// output: 1
```
### Proměnná *const*
Lokální proměnné s platností v bloku kde je deklarována. Hodnotu konstanty nelze změnit pomocí operátoru přiřazení, ale pokud je konstanta objektem, lze k ní přidávat, aktualizovat nebo odstraňovat vlastnosti.
```js
const number = 42;

try {
  number = 99;
} catch (err) {
  console.log(err);
  // output: TypeError: neplatné přiřazení proměnné typu const
  // přesný výstup se může lišit v závislosti na prohlížeči
}

console.log(number);
// output: 42

```
### Proměnná *var*
Proměnné s platností v rámci funkce nebo globální proměnné. Volitelně lze každou z nich inicializovat určitou hodnotou.

```js
var x = 1;

if (x === 1) {
  var x = 2;
  console.log(x);
  // output: 2
}
console.log(x);
// output: 2
```
## Funkce
Funkce slouží k seskupení kódu pro opakované použití.
```js
function pozdrav(jmeno){
    return "Ahoj " + jmeno;
}

console.log(pozdrav("Petr"));
console.log(pozdrav("Lucie"));
```

## DOM (Document Object Model)
DOM je **objektová reprezentace HTML dokumentu**, kterou lze ovládat pomocí JavaScriptu.
A.K.A. 
Je to strom objektů, kde každý HTML element je objekt, se kterým může JavaScript pracovat.

Příklad DOM stromu:
```html
<body>
  <h1>Nadpis</h1>
  <p>Text odstavce</p>
  <button id="btn">Klikni</button>
</body>
```
`<body>` → objekt

`<h1>` → objekt uvnitř `<body>`

`<p>` → objekt uvnitř `<body>`

`<button>` → objekt uvnitř `<body>`

JavaScript může tyto objekty **vyhledat**, **přečíst obsah**, **změnit obsah** nebo **změnit vzhled**.


## Výběr elementů
1. Podle ID (jedinečné označení)
    ```js
    let tlacitko = document.getElementById("btn");
    ```
2. Podle třídy (může být víc elementů stejné třídy)
    ```js
    let odstavce = document.getElementsByClassName("paragraf");
    ```
3. CSS selektor (moderní způsob)
    ```js
    let prvniOdstavec = document.querySelector("p");       // první odstavce
    let vsechnyOdstavce = document.querySelectorAll("p");  // všechny odstavce
    ```

### Události
Události umožňují reagovat na akce uživatele.

Například:
- click – kliknutí myší  
- mouseover – přejetí myší  
- submit – odeslání formuláře

Některé události:
- **click - kliknutí myší**
- dblclick - dvojklik
- **mouseover - najedeš myší na element**
- mouseout - myš opustí element
- keydown - stiskneš klávesu
- keyup - pustíš klávesu
- **submit - odeslání formuláře**

Registrace události (pomocí Listeneru):
```js
element.addEventListener("click", function(){
    alert("Kliknuto!");
});
```
Alternativa – inline atribut:
```html
<button onclick="alert('Klik!')">Klikni</button>
```

### Manipulace
Každý element na stránce je DOM objekt, takže ho můžeme měnit.
#### Manipulace HTML
```js
element.innerHTML = "Nový text";    // změní obsah elementu
element.innerHTML += "<b> A toto je tučně!</b>"; // přidá ztučnění do odstavce
element.textContent = "Jen text";   // změní pouze text

let obrazek = document.getElementById("foto");
obrazek.src = "novy_obrazek.jpg";  // změní zdroj obrázku
obrazek.alt = "Nový popis obrázku";

// Přidání nového elementu
let novyParagraf = document.createElement("p");
    novyParagraf.innerHTML = "Nový odstavec";
    document.body.appendChild(novyParagraf);   // přidá do stránky
```
#### Manipulace CSS:
Každý element má styl, který můžeme měnit přes JavaScript.
```js
element.style.color = "red";         // změní barvu textu
element.style.backgroundColor = "yellow";  // změní pozadí
element.classList.add("nova-trida");       // přidání CSS třídy
element.classList.remove("stara-trida");  // odebrání CSS třídy
```
