## CSS – význam
**CSS (Cascading Style Sheets)** určuje vzhled webové stránky.
Odděluje:
- obsah (HTML)
- vzhled (CSS)

Výhody:
- přehlednost
- znovupoužitelnost stylů
- responzivní design

## Způsoby připojení CSS
**Externí soubor**
```html
<link rel="stylesheet" href="style.css">
```
**Interní styl**
```html
<style>
p {color:red;}
</style>
```
**Inline styl**
```html
<p style="color:red;">
```
## CSS pravidlo
```css
selektor {
vlastnost: hodnota;
}
```
Příklad:
```css
p {
color: blue;
font-size: 16px;
}
```

## Box model
Každý prvek má:
- content
- padding
- border
- margin

To určuje velikost a rozestupy prvků.
## Rozmístění prvků
**float**
- obtékání

**position**
- static
- relative
- absolute
- fixed

**display**
- block
- inline
- flex
- grid

## Responzivní design
Responzivní design umožňuje správné zobrazení na mobilu.
**Viewport**
```html
<meta name="viewport">
```
**Media query**
```css
@media (max-width:600px)
```
Používá se pro různé velikosti obrazovky.
