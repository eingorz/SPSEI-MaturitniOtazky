## CSS – význam
**CSS (Cascading Style Sheets)** určuje vzhled webové stránky. V překladu **kaskádové styly**, kaskádové protože se na sebe mohou vrstvit definice stylů, ale platí jenom ta poslední.

**Odděluje vzhled (CSS) od obsah (HTML).**

Výhody:
- přehlednost
- znovupoužitelnost stylů
- responzivní design

## Způsoby připojení CSS
**Externí soubor** - připojení stylového souboru externě, tímpádem **může být připojen k více stránkám** => nejpoužívanější
```html
<link rel="stylesheet" href="style.css">
```
**Interní styl** - vlastně obsah css souboru **v hlavičce stránky**
```html
<style>
p {color:red;}
</style>
```
**Inline styl** - styl přímo jako **atribut tagu**
```html
<p style="color:red;">
```

Z pojmu "kaskádové styly" je k připojení určitá hierarchie (platí jen ta poslední definice stylu).

Od nejvyšší priority po nejnižší:
1. Inline
2. Interní
3. Externí

## Syntaxe CSS
Selektorem může být i tag. Nějaké vlastnosti mohou být zkrácené.
```css
selektor {
vlastnost: hodnota;
}
```
Příklad:
```css
p {
color: blue;
font-size: 16px;
}
```
### Zkrácené vlastnosti
Takzvaně *shorthand properties* jsou vlastnosti, které umožňují nastavit hodnoty dalších vlastností CSS v jedné deklaraci. Tyhle vlastnosti se dají poznat sjednocenou syntaxí `vlastnost:`, když její podrobnosti jsou ve formě `vlastnost-podrobnost`, samozřejmě pokud má vlastnost více podrobnější podrobnost (`vlastnost-podrobnost-podrobnejsi`), tak syntaxi by měla `vlastnost-podrobnost:` (Pochopte z příkladů).

Příklad:
```css
body{
  background-color: red; /* nastavení barvy pozadí na červenou */
  background: url("images/bg.gif") no-repeat left top; /* nastavení pozadí na .gif, bez opakování a zarovnání na levý horní roh*/
  /* v tenhle moment se barva pozadní resetuje na default - průhlednou */
}
```

U jakékoli zkrácené vlastnosti se každá jednotlivá hodnota, která není zadána, nastaví na odpovídající počáteční hodnotu (případně přepíše hodnoty dříve nastavené pomocí nezkrácených vlastností).

#### Záludy případy
1. **Vynechání vlastnosti:**
  - Hodnota, která není zadána, se nastaví na výchozí hodnotu definovanou zkrácenou vlastností, která se může lišit od počáteční hodnoty vlastnosti (**přepíše dříve nastavené hodnoty**), viz. dřívější příklad
2. **Řazení vlastností:**
  - Zkrácené vlastnosti se snaží nevynucovat určité pořadí hodnot vlastností, které nahrazují. Funguje dobře, když tyto vlastnosti používají hodnoty různých typů, protože pořadí nemá žádný význam, ale to nefunguje tak snadno, když několik vlastností může mít stejné hodnoty. (**border-style, margin, padding, border-radius**)


##### Strany a hrany box modelu - syntax (border-style, margin, padding, border-radius)
###### Strany
- 1 hodnota = **všechny strany**
- 2 hodnoty = první hodnota ovlivňuje **horní a spodní stranu**, druhá hodnota **levou a pravou stranu**
- 3 hodnoty = první hodnota ovlivňuje **horní  stranu**, druhá hodnota **levou a pravou stranu**, třetí hodnota **spodní stranu**
- 4 hodnoty = **horní-pravá-spodní-levá** strana

###### Hrany
- 1 hodnota = **všechny hrany**
- 2 hodnoty = první hodnota ovlivňuje **horní-levý a spodní-pravý roh**, druhá hodnota **horní-pravý a spodní-levý roh**
- 3 hodnoty = první hodnota ovlivňuje **horní-levý roh**, druhá hodnota **horní-pravý a spodní-levý roh**, třetí hodnota **spodní-pravý roh**
- 4 hodnoty = **horní-levý; horní-pravý; spodní-pravý; spodní-levý**

## Selektory
### Potomek (mezera)
Upravuje všechny prvky uvnitř jiného prvku (na libovolné úrovni):
```css
div p {  
  color: red;  
}
```
všechny `<p>` uvnitř `<div>`

### Třída (.)
Upravuje všechny prvky s danou třídou (atribut `class = "..."`):

```css
.container {  
  background-color: azure;  
}
```
všechny prvky s třídou `container`

### Identifikátor (#)
Upravuje prvek s daným **id** (atribut `id = "..."`)

```css
#hlavni {  
  color: crimson;  
}
```
jediný prvek s id `hlavni`

### Třída (.)
Upravuje všechny prvky s danou třídou (atribut `class = "..."`)

```css
.container {  
  background-color: azure;  
}
```
všechny prvky s třídou `container`

### Potomek (mezera)
Upravuje všechny prvky uvnitř jiného prvku (na libovolné úrovni)
```css
div p {  
  color: red;  
}
```
všechny `<p>` uvnitř `<div>`
### Přímý potomek (`>`)
Upravuje jen prvky, které jsou **přímo uvnitř**
```css
div > p {  
  color: blue;  
}
```
jen `<p>`, které jsou **hned uvnitř `<div>`** (ne hlubší)
### Sourozenec (`+`)
Upravuje **první prvek hned za jiným**
```css
h1 + p {  
  color: green;  
}
```
první `<p>` hned za `<h1>`

### Obecný sourozenec (`~`)
Upravuje **všechny sourozence za prvkem**
```css
h1 ~ p {  
  color: orange;  
}
```
všechny `<p>` za `<h1>` na stejné úrovni

### Univerzální selektor (`*`)
Upravuje úplně všechno
```css
* {  
  margin: 0;  
}
```

#### Typový
Upravuje všechny prvky elementu:
```css
p {
	padding-left: 10px;
}
```
všechny `<p>` kdekoliv v souboru
### Atributový selektor (`[ ]`)
Upravuje podle atributu
```css
input[type="text"] {  
  border: 1px solid black;  
}
```

### Více tříd najednou
Prvek musí mít **obě třídy**
```css
.button.primary {  
  background: blue;  
}
```

### Více selektorů (čárka `,`)
Použije styl na více prvků
```css
h1, h2, h3 {  
  color: black;  
}
```

## Pseudoelementy

```css
}  
  
p::before {  
  content: "-> ";  
}
```

## Responzivní design
Responzivní design umožňuje správné zobrazení na mobilu.
**Viewport**
```html
<meta name="viewport">
```
**Media query**
```css
@media (max-width:600px)
```
Používá se pro různé velikosti obrazovky.
