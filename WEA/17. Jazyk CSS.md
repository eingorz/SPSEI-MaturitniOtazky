## CSS – význam
**CSS (Cascading Style Sheets)** určuje vzhled webové stránky.
Odděluje:
- obsah (HTML)
- vzhled (CSS)

Výhody:
- přehlednost
- znovupoužitelnost stylů
- responzivní design

## Způsoby připojení CSS
**Externí soubor**
```html
<link rel="stylesheet" href="style.css">
```
**Interní styl**
```html
<style>
p {color:red;}
</style>
```
**Inline styl**
```html
<p style="color:red;">
```
## CSS pravidlo
```css
selektor {
vlastnost: hodnota;
}
```
Příklad:
```css
p {
color: blue;
font-size: 16px;
}
```

## Selektory
### Potomek (mezera)
Upravuje všechny prvky uvnitř jiného prvku (na libovolné úrovni):
```css
div p {  
  color: red;  
}
```
všechny `<p>` uvnitř `<div>`

### Třída (.)
Upravuje všechny prvky s danou třídou (atribut `class = "..."`):

```css
.container {  
  background-color: azure;  
}
```
všechny prvky s třídou `container`

### Identifikátor (#)
Upravuje prvek s daným **id** (atribut `id = "..."`)

```css
#hlavni {  
  color: crimson;  
}
```
jediný prvek s id `hlavni`

### Třída (.)
Upravuje všechny prvky s danou třídou (atribut `class = "..."`)

```css
.container {  
  background-color: azure;  
}
```
všechny prvky s třídou `container`

### Potomek (mezera)
Upravuje všechny prvky uvnitř jiného prvku (na libovolné úrovni)
```css
div p {  
  color: red;  
}
```
všechny `<p>` uvnitř `<div>`
### Přímý potomek (`>`)
Upravuje jen prvky, které jsou **přímo uvnitř**
```css
div > p {  
  color: blue;  
}
```
jen `<p>`, které jsou **hned uvnitř `<div>`** (ne hlubší)
### Sourozenec (`+`)
Upravuje **první prvek hned za jiným**
```css
h1 + p {  
  color: green;  
}
```
první `<p>` hned za `<h1>`

### Obecný sourozenec (`~`)
Upravuje **všechny sourozence za prvkem**
```css
h1 ~ p {  
  color: orange;  
}
```
všechny `<p>` za `<h1>` na stejné úrovni

### Univerzální selektor (`*`)
Upravuje úplně všechno
```css
* {  
  margin: 0;  
}
```

#### Typový
Upravuje všechny prvky elementu:
```css
p {
	padding-left: 10px;
}
```
všechny `<p>` kdekoliv v souboru
### Atributový selektor (`[ ]`)
Upravuje podle atributu
```css
input[type="text"] {  
  border: 1px solid black;  
}
```

### Více tříd najednou
Prvek musí mít **obě třídy**
```css
.button.primary {  
  background: blue;  
}
```

### Více selektorů (čárka `,`)
Použije styl na více prvků
```css
h1, h2, h3 {  
  color: black;  
}
```

## Pseudotřídy
a:hover     /* při najetí myší */  
a:active    /* při kliknutí */  
a:visited   /* navštívený odkaz */  
input:focus /* když je vybraný */

## Pseudoelementy
Pracují s částí prvku:
```css
p::first-letter {  
  font-size: 200%;  
}  
  
p::before {  
  content: "-> ";  
}
```
## Základní vlastnosti
**Barva**
- color
- background-color

**Font**
- font-family
- font-size
- font-weight

**Pozadí**
- background-image

## Box model
Každý prvek má:
- content
- padding
- border
- margin

To určuje velikost a rozestupy prvků.
## Rozmístění prvků
**float**
- obtékání

**position**
- static
- relative
- absolute
- fixed

**display**
- block
- inline
- flex
- grid

## Responzivní design
Responzivní design umožňuje správné zobrazení na mobilu.
**Viewport**
```html
<meta name="viewport">
```
**Media query**
```css
@media (max-width:600px)
```
Používá se pro různé velikosti obrazovky.
