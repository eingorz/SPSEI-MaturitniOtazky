# Webová aplikace
- aplikace na webovém serveru (na internetu/intranetu)
- nepotřebuje se nainstalovat, přistupuje se k ní pomocí webového prohlížeče
- připojení přes http/https (Hyper Text Transfer Protocol (Secure))

Umožňuje:
- práci s daty (např. formuláře, databáze)
- interakci s uživatelem
- dynamické generování obsahu

## Rozdíl mezi webovou stránkou a webovou aplikací
Webová stránka:
- slouží hlavně k zobrazení informací
- většinou statická
- minimální interakce

Webová aplikace:
- umožňuje aktivní práci s daty
- je dynamická
- obsahuje aplikační logiku
- reaguje na vstupy uživatele

## Funkčnost
1. klient(prohlížeč x uživatel) zadá URL adresu
    - pomocí DNS zjistí ip adresu serveru a naváže spojení
2. klient pošle požadavek na server (HTTP/HTTPS)
3. zpracování požadavku serverem
4. server vrátí odpověď klientovi (HTML stránku, JSON, ...)
5. prohlížeč zobrazí klientovi výsledek

### Struktura (vrstvy)
1. prezentační(**frontend**):
    - běží ve webovém prohlížeči
    - zajišťuje zobrazení dat a interakci s uživatelem
2. logická(**backend**):
    - zpracování logiky aplikace a požadavky
    - komunikace s databází
3. datová:
    - databáze, ukládání dat

# Architektury
- určení jak je aplikace rozvržena (pro developera)

## MVC (Model View Controller) architektura
- rozdělení podle struktury
- přehledná, vhodná pro větší aplikace, oddělení logiky od vzhledu
- **View** => prezentační vrstva
- **Controller** => logická vrstva
- **Model** => datová vrstva

Výhody:
- přehlednost
- snadná údržba
- vhodné pro větší projekty

## Monolitická architektura
- celá aplikace jako jeden celek
- pro malé aplikace (obtížnější škálování/rozšiřování)

## Mikroservisní architektura
- rozdělení na nezávislé malé služby
- modulární, škálovatelná, technicky náročná
- Rest API

## Jazyky
### Frontend
- HTML (Hyper Text Markup Language) - základ, definuje obsah
- CSS (Cascading Style Sheet) - stylování stránky
- JavaScript - interaktivnost, dynamická změna obsahu
- Frameworky - React, Angular, Vue

### Backend
- zpracování logiky aplikace
- Jazyky:
    - PHP, Java, C#, Python, JavaScript ve formě Node.js
- Frameworky:
    - Laravel(PHP), Spring(Java), ASP.NET(C#), Django(Python), Flask(Python), Express(Node.js)

### Databáze
1. Relační (SQL)
	- data v tabulkách (sloupce a řádky)
	- mezi tabulkami existují relace (vztahy)
	- MySQL, PostrgreSQL, Oracle, MariaDB
2. Nerelační (NoSQL)
	- není pevně daná tabulková struktura
	- flexibilnějí, pro velké objemy dat
	- Základní typy:
		1. Dokumentové - JSON (MongoDB, ...)
		2. Klíč-hodnota - páry dat (Key -> value, Redis)
