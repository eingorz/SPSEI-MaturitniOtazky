## Webová aplikace – pojem
Webová aplikace je program běžící na serveru a dostupný přes webový prohlížeč.
Příklady:
- internetové bankovnictví
- e-shop
- sociální sítě

## Princip fungování
1. uživatel odešle požadavek  
2. server zpracuje logiku  
3. server odešle odpověď
