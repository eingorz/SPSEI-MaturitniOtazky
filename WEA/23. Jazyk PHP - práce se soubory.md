# Procházení souborů
Webová aplikace může číst obsah složek a pracovat se soubory.
### mkdir()
- vytvoření složky
    ```php
    mkdir("data");  // vytvoří složku 'data' v aktuálním adresáři
    ```

### file_exists()
- kontrola existence souboru
    ```php
    if (file_exists("test.txt")) {
    echo "Soubor existuje";
    } else {
        echo "Soubor neexistuje";
    }
    ```

### opendir() a readdir()
- procházení adresáře
    ```php
    $dir = opendir("soubory");
    while (($file = readdir($dir)) !== false) {
        if ($file != "." && $file != "..") {
            echo $file . "<br>";
        }
    }
    closedir($dir);
    ```

### glob()
- vyhledávání souborů podle masky
    ```php
    $images = glob("*.jpg"); // vrátí pole všech jpg souborů v aktuálním adresáři
    foreach($images as $img) {
        echo $img . "<br>";
    }
    ```

## Upload souborů
Upload znamená **nahrání souboru z počítače uživatele na server**.
Používá se například:
- u profilových fotek
- u dokumentů
- u příloh

# Nastavení formuláře
Formulář **musí obsahovat** `method="post"` a `enctype="multipart/form-data"`:
```html
<form method="post" enctype="multipart/form-data">
    <input type="file" name="soubor">
    <input type="submit" value="Odeslat">
</form>
```
## Pole $_FILES
Po odeslání formuláře jsou informace o souboru dostupné v poli `$_FILES`:
Struktura:
```php
$_FILES["soubor"]["name"]       // originální název souboru
$_FILES["soubor"]["tmp_name"]   // dočasné uložiště na serveru
$_FILES["soubor"]["size"]       // velikost souboru v bytech
$_FILES["soubor"]["type"]       // MIME (Multipurpose Internet Mail Extensions - internetový standard označení formátu souboru) typ souboru
```
## Přesunutí souboru na server
### move_uploaded_file()
Funkce slouží k přesunu souboru z dočasného adresáře PHP na cílové místo na serveru.

Syntax:
```php
bool move_uploaded_file(string $tmp_name, string $destination)
```

`$tmp_name` – cesta k dočasnému souboru, který PHP vytvořilo při uploadu `($_FILES["soubor"]["tmp_name"])`
`$destination` – cílová cesta a název souboru na serveru

Funkce vrací **true**, pokud přesun proběhne úspěšně, nebo **false**, pokud se něco nepovede.

Příklad bezpečného přesunu:
```php
$upload_dir = "uploads/";

// vytvoření složky, pokud neexistuje
if (!file_exists($upload_dir)) {
    mkdir($upload_dir, 0755, true);
}

// unikátní název souboru (aby se nepřepsal)
$filename = uniqid() . "_" . basename($_FILES["soubor"]["name"]);
$destination = $upload_dir . $filename;

// kontrola velikosti a typu souboru
$max_size = 2 * 1024 * 1024; // 2 MB
$allowed_types = ["image/jpeg", "image/png", "application/pdf"];

if ($_FILES["soubor"]["size"] <= $max_size && in_array($_FILES["soubor"]["type"], $allowed_types)) {
    if (move_uploaded_file($_FILES["soubor"]["tmp_name"], $destination)) {
        echo "Soubor byl nahrán: $filename";
    } else {
        echo "Chyba při nahrávání souboru.";
    }
} else {
    echo "Soubor je příliš velký nebo nepovoleného typu.";
}
```
