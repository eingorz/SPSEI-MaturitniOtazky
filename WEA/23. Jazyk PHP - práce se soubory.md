# Procházení souborů
Webová aplikace může číst obsah složek.
### mkdir()
vytvoření složky
```php
mkdir("data");
```
### file_exists()
kontrola existence souboru
```php
file_exists("test.txt")
```
### opendir() a readdir()
procházení adresáře
```php
$dir = opendir("soubory");
```
### glob()
vyhledávání souborů podle masky
```php
glob("*.jpg");
```
## Upload souborů
Upload znamená **nahrání souboru z počítače uživatele na server**.
Používá se například:
- u profilových fotek
- u dokumentů
- u příloh

## Nastavení formuláře
Formulář musí obsahovat:
```html
<form  method="post"  enctype="multipart/form-data">
```
Input:
```html
<input  type="file">
```
## Pole $_FILES
Struktura:
```php
$_FILES["soubor"]["name"]  
$_FILES["soubor"]["tmp_name"]  
$_FILES["soubor"]["size"]  
$_FILES["soubor"]["type"]
```
## Funkce pro upload
### move_uploaded_file()
přesune soubor na server
```php
move_uploaded_file()
```
