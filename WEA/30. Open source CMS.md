# CMS
CMS neboli **Content Management System** slouží k správě obsahu webu.
Umožňuje vytvářet webové stránky *bez nutnosti programovaní*.

**Open Source** znamená, že zdrojová kód je *volně dostupný* a kdokoliv ho může upravovat, šířit a vylepšovat.

Bývá používán pro:
- firmy
- školy
- e-shopy
- blogy

## Výhody 
- zdarma (není potřeba kupovat licenci)
- možnost úprav kódu
- rozšiřitelnost (pluginy)
- rychlé vytvoření webu

## Nevýhody
- bezpečnostní rizika
- nutnost aktualizací
- závislost na pluginech
- omezená flexibilita

### Nejznámější Open Source CMS
- WordPress
- Joomla
- Drupal

# WordPress
Nejpoužívanější CMS na světě.
Vhodný pro:
- blogy
- firemní weby
- e-shopy

Používá:
- PHP
- MySQL

## Základní činnosti
- vytváření článků (Posts)
- stránky (Pages)
- šablony (Themes) - vzhled stránek
- správa uživatelů - role a práva k webu
- menu a navigace
- pluginy

### Pluginy
Pluginy rozšiřují jednotlivé funkce wordpressu.
Jsou *modulární*, rozšiřují jednotlivé části (galerie, formuláře, e-shopy, zálohy, zabezpečení, design, ...)

Příklady pluginů:
- Photo Gallery
- NextGen Gallery
- Contact Form 7
- WooCommerce
- Yoast SEO
- Elementor
- Wordfence

# Joomla
1. Správa obsahu
- tvorba článků (Articles) - psaní textu, vkládání obrázků/videí
- kategorizování - organizace článků do struktur

2. Správa menu
- vytváření navigace webu
- propojení s články, kategoriemi a komponentami

3. Šablony (Templates)
- změna vzhledu webu
- možnost instalace šablon nebo úpravy CSS

4. Uživatelské účty
- správa uživatelů
- role (administrátor, editor, registrovaný uživatel)

5. Moduly
- menší části stránky
- menu, přihlášení, vyhledávání

6. Komponenty
- hlavní funkční části systému
- správa článků, e-shop, galerie

7. Instalace rozšíření
- jen přes administraci Joomla
- pluginy, moduly, komponenty

## Pluginy
Joomla používá více typů rozšíření:
- pluginy
- moduly
- komponenty

### Pluginy (Plugins)
- malé funkce na pozadí
- reagují na události (např. uložení článku)

Příklady:
- SEO pluginy
- autentizace uživatelů

#### Výhody pluginů
- rozšíření funkcionality
- není nutné programovat
- velké množství dostupných doplňků
#### Nevýhody pluginů
- některé jsou placené
- mohou zpomalit web
- nutnost aktualizací (bezpečnost)

##### Nejznámější rozšíření
- VirtueMart → e-shop
- Akeeba Backup → zálohování webu
- JCE Editor → pokročilý editor obsahu
- RSForm → tvorba formulářů
