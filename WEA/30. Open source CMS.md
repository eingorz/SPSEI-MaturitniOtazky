# CMS
CMS je **Content Management System**. Open source znamená, že zdrojový kód je zdarma a volně přístupný.
Umožňuje správu webu bez programování.
### Výhody
- rychlá tvorba webu
- jednoduchá správa
- mnoho pluginů
### Nevýhody
- bezpečnostní rizika
- pomalejší
- omezená flexibilita
### Nejznámější CMS
- WordPress
- Joomla
- Drupal
## WordPress
Nejpoužívanější CMS na světě.

Používá:
- PHP
- MySQL
### Základní činnosti
- instalace
- správa článků
- správa uživatelů
- instalace pluginů
### Pluginy
Pluginy rozšiřují funkce WordPressu.
Například:
- SEO plugin
- WForms
- galerie
- e-shop
