## Formulářové prvky
Formuláře slouží ke **sběru dat od uživatele**.
Použití:
- registrace
- přihlášení
- objednávky
- vyhledávání

## Prvek form
```html
<form action="zpracovani.php" method="post">  
</form>
```
## Input a jeho typy
```html
<input type="text">  
<input type="password">  
<input type="email">  
<input type="number">  
<input type="date">  
<input type="checkbox">  
<input type="radio">  
<input type="file">  
<input type="submit">
```
## Další formulářové prvky
**textarea**
- víceřádkový text

**select**
- výběr ze seznamu

**option**
- položky seznamu

**button**
- tlačítko

## Atributy
**name**
- název proměnné

**value**
- hodnota

**placeholder**
- nápověda

**required**
- povinné pole
