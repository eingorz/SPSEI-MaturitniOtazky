# Autentizace
Autentizace znamená **ověření identity uživatele**.
Například:
- přihlášení do systému
- administrace webu
## Princip
1. uživatel zadá login a heslo  
2. server zkontroluje databázi  
3. vytvoří session
## Session
Session ukládá data mezi stránkami.
Začíná funkcí:
```php
session_start();
```
## Pole $_SESSION
```php
$_SESSION["uzivatel"] = "admin";
```
## Funkce
### unset()
smaže proměnnou
```php
unset($_SESSION["uzivatel"]);
```
## Ukládání hesel
Hesla se nikdy neukládají přímo.
Používá se **hashování** a popřípadně i *sůl*.
Například:
```php
password_hash()  
password_verify()
```
