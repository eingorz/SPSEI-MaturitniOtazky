# Význam knihoven
Knihovny jsou **hotové balíčky funkcí**, které usnadňují práci.
Výhody:
- rychlejší vývoj
- méně kódu
- hotové funkce

Nevýhody:
- větší velikost stránky
- závislost na knihovně
## CDN
CDN je síť serverů poskytující knihovny.
Například:
```html
<script src="https://cdn..."></script>
```
Výhoda:
-   rychlé načítání
## jQuery
Jedna z nejznámějších JS knihoven.
Zjednodušuje:
- manipulaci DOM
- události
- animace
- AJAX
### Syntaxe
```js
$(selector).action()
```
### Příklad
```js
$("#btn").click(function(){  
alert("Ahoj");  
});
```
## Pluginy
Plugin je **rozšíření knihovny**.
Například:
- slider obrázků
- galerie
- validace formuláře
