# Význam knihoven
Knihovny jsou **hotové balíčky funkcí**, které usnadňují práci a urychlují vývoj webových aplikací(JavaScript).
Výhody:
- rychlejší vývoj – nemusíte programovat vše od začátku
- méně kódu – hotové funkce a moduly
- spolehlivost – většina knihoven je dobře otestovaná

Nevýhody:
- větší velikost stránky – načítání dalších souborů může zpomalit web
- závislost na knihovně – změny nebo zastarání knihovny mohou ovlivnit váš projekt
- potenciální konflikty mezi pluginy

## Připojení knihoven
Knihovny a pluginy lze připojit dvěma způsoby:
1. Stažením souboru a umístěním na server
    ```html
    <script src="js/nazev_knihovny.js"></script>
    ```
2. Pomocí **CDN** (Content Delivery Network)
    CDN je síť serverů, která poskytuje knihovny rychleji a efektivněji.
    ```html
    <script src="https://cdn.nazevknihovny.com/knihovna.min.js"></script>
    ```
    Výhoda CDN:
    - Rychlé načítání díky geograficky rozprostřeným serverům

# Knihovna jQuery
Jedna z nejznámějších a nejpoužívanějších JavaScriptových knihoven.

Zjednodušuje:
- Manipulaci s DOM (vyhledávání a změny elementů)
- Práci s událostmi (click, hover, atd.)
- Animace
- AJAX (asynchronní načítání dat bez reloadu stránky)

## Syntaxe
```js
$(selector).action()
```

### Příklad použití
```js
// Po kliknutí na tlačítko se zobrazí alert
$("#btn").click(function(){  
    alert("Ahoj");  
});
```

# Pluginy
Pluginy jsou **rozšíření knihoven**, která přidávají nové možnosti bez nutnosti psát vlastní kód od nuly.
Příklady pluginů:
- Slider obrázků (např. Slick Slider, Swiper)
- Galerie obrázků (např. Lightbox)
- Validace formuláře (např. jQuery Validation)
- Kalendáře a plánovače (např. FullCalendar)

## Připojení pluginu
Plugin se obvykle připojuje po načtení hlavní knihovny:
```html
<!-- jQuery -->
<script src="https://code.jquery.com/jquery-3.6.0.min.js"></script>
<!-- Plugin -->
<script src="https://cdn.example.com/plugin.min.js"></script>
```
