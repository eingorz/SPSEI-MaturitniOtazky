## Internet – vznik, vývoj, služby
- celosvětová síť propojených počítačových sítí.
- Vznikl v 60. letech v USA jako projekt **ARPANET** financovaný armádou.
- Cílem bylo vytvořit síť odolnou proti výpadkům.
- Později se síť rozšířila na univerzity a výzkumné instituce.

**Vývoj:**
- **1969** – ARPANET
- **1983** – zavedení protokolu TCP/IP
- **1990** – vznik WWW

**Hlavní služby internetu:**
- **WWW** – webové stránky
- **E-mail** – elektronická pošta
- **FTP** – přenos souborů
- **VoIP** – internetové telefonování
- cloudové služby
- streamování videa

## Služba WWW – vznik, vývoj, pojmy, princip
- **WWW (World Wide Web)** je služba internetu umožňující zobrazování hypertextových dokumentů.

**Vznik:**
- vytvořil **Tim Berners-Lee** v roce 1989 v CERNu.

### Základní pojmy:
**Webový server**
- počítač nebo software, který poskytuje webové stránky
- např. Apache, Nginx

**Webový prohlížeč**
- program pro zobrazování webových stránek
- např. Chrome, Firefox, Edge

**HTTP**
- protokol pro komunikaci mezi klientem a serverem
- klient odešle požadavek a server vrátí odpověď

**URL**
- adresa webové stránky
- např.
`https://www.example.com/index.html`

**Struktura:**
- protokol
- doména
- cesta k souboru

## Princip fungování WWW
1.  uživatel zadá URL do prohlížeče
2.  prohlížeč odešle HTTP požadavek na server
3.  server pošle HTML dokument
4.  prohlížeč stránku vykreslí

Pokud stránka obsahuje:
- CSS
- JavaScript
- obrázky

prohlížeč je načte zvlášť.
## Webové standardy
Web používá několik základních technologií:
**HTML**
- struktura stránky

**CSS**
- vzhled stránky

**JavaScript**
- interaktivita

**Formáty médií**
- obrázky: JPG, PNG, SVG
- video: MP4
- audio: MP3

## Konsorcium W3C
- **W3C (World Wide Web Consortium)** je organizace vytvářející webové standardy.
- založeno v roce 1994 Timem Berners-Lee.

Cílem je:
- jednotné fungování webu  
- kompatibilita mezi prohlížeči
- vývoj standardů HTML, CSS

## Jazyk HTML – význam, verze, syntaxe
**HTML (HyperText Markup Language)** je značkovací jazyk pro tvorbu webových stránek.
Používá **tagy (značky)**.
Například:
```html
<p>Toto je odstavec</p>
```
**Vývoj verzí:**
- HTML 1.0
- HTML 4
- XHTML
- HTML5 (dnes používané)

## Základní struktura HTML dokumentu
```html
<!DOCTYPE html>  
<html>
	<head>
		<meta  charset="UTF-8">
		<title>Název stránky</title>
	</head>
	<body>
		Obsah stránky  
	</body>
</html>
```
**Důležité části:**
`DOCTYPE`
- určuje typ dokumentu

`head`
- metadata stránky

`meta charset`
- znaková sada (např. UTF-8)

`body`
- obsah stránky

## Bloková struktura HTML
**Obecné bloky**
- `div`
- `span`

**HTML5 bloky**
- `header`
- `nav`
- `main`
- `section`
- `article`
- `footer`

Výhoda:
- lepší struktura
- lepší SEO

## Strukturování textu
**Nadpisy**
```html
<h1> až <h6>
```
**Odstavce**
```html
<p>
```
**Seznamy**
Nečíslovaný:
```html
<ul>  
<li></li>  
</ul>
```
Číslovaný:
```html
<ol>  
<li></li>  
</ol>
```
**Tabulky**
```html
<table>  
<tr>  
<td>
```
**Zvýraznění textu**
- `strong`
- `em`
- `b`
- `i`

## Odkazy
Značka:
```html
<a href="adresa">text</a>
```
**Atributy**
- `href` – cílová adresa
- `target="_blank"` – otevře v nové kartě

**Typy odkazů**
- externí
- interní
- kotva v dokumentu
- email

## Média
**Obrázky**
```html
<img src="obrazek.jpg" alt="popis">
```
**Audio**
```html
<audio controls>
```
**Video**
```html
<video controls>
```
## Entity
Entity jsou speciální znaky zapisované pomocí kódu.
Příklady:
`&lt;   <`
`&gt;   >`
`&nbsp;   mezera`
`&copy;   ©`
Používají se pro znaky, které mají v HTML speciální význam.
## Validace
Validace kontroluje, zda je HTML napsáno správně podle standardu.
Význam:
- správné zobrazení
- kompatibilita
- lepší SEO

Postup:
1. otevřít validátor W3C
2. vložit URL nebo kód
3. opravit chyby
