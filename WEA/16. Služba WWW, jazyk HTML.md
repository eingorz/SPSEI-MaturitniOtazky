## Internet – vznik, vývoj, služby
- celosvětová síť propojených počítačových sítí.
- Vznikl v 60. letech v USA jako projekt **ARPANET** financovaný armádou.
- Cílem bylo vytvořit síť odolnou proti výpadkům.
- Později se síť rozšířila na univerzity a výzkumné instituce.

**Vývoj:**
- **1969** – ARPANET
- **1983** – zavedení protokolu TCP/IP
- **1990** – vznik WWW

**Hlavní služby internetu:**
- **WWW** – webové stránky
- **E-mail** – elektronická pošta
- **FTP** – přenos souborů
- **VoIP** – internetové telefonování
- cloudové služby
- streamování videa

## Služba WWW – vznik, vývoj, pojmy, princip
**WWW (World Wide Web)** je služba internetu umožňující zobrazování hypertextových dokumentů.

**Vznik:**
- navrl **Tim Berners-Lee** v roce 1989 v CERNu a první funkční verze vznikla v roce 1990.

### Základní pojmy:
**Webový server**
- počítač nebo software, který poskytuje webové stránky
- např. Apache, Nginx

**Webový prohlížeč**
- program pro zobrazování webových stránek
- např. Chrome, Firefox, Edge

**HTTP**
- protokol pro komunikaci mezi klientem a serverem
- klient odešle požadavek a server vrátí odpověď

**URL**
- adresa webové stránky
- např.
`https://www.example.com/index.html`

**Struktura:**
- protokol
- doména
- cesta k souboru

## Princip fungování WWW
1.  uživatel zadá URL do prohlížeče
2.  prohlížeč odešle HTTP požadavek na server
3.  server pošle HTML dokument
4.  prohlížeč stránku vykreslí

Pokud stránka obsahuje:
- CSS
- JavaScript
- obrázky

Prohlížeč je načte zvlášť.

## Webové standardy

**HTML** - struktura stránky

**Formáty médií**

## Konsorcium W3C
- **W3C (World Wide Web Consortium)** je mezinárodní organizace, která se zabývá vývojem jazyka a vydává standardy syntaxe HTML a CSS, čímž zajišťuje jednotu všech internetových prohlížečů.
- založena v roce 1994 Timem Berners-Lee.

Cílem je:
- jednotné fungování webu  
- kompatibilita mezi prohlížeči
- vývoj standardů HTML, CSS

# Jazyk HTML – význam, verze, syntaxe
**HTML (HyperText Markup Language)** je značkovací jazyk pro tvorbu webových stránek.
Používá tzv. **tagy**.

Například:
```HTML
<p>Toto je odstavec</p>
```
**Vývoj verzí:**
- HTML 1.0
- HTML 4
- XHTML
- HTML5 (dnes používané)

## Syntaxe tagů
HTML značky (**tagy**) jsou uzavřeny ve špičatých závorkách `< >`. Každá značka má přesnou syntaxi. **Zapíšeme-li značku chybně** (neexistující značku), bude ji prohlížeč ignorovat (**nedetekuje chybu**). Text v ní umístěný se pak může ocitnout v jiném místě stránky či být jinak formátovaný, neboť není ohraničen správnou značkou. Některé chyby prohlížeče sami opraví. Nemůžeme však spoléhat, že všechny prohlížeče doplní chybějící kód stejně.

HTML tagy se rozdělují na **párové** a **nepárové**. Obsahem tagu může být cokoliv, klidně i další značky, které text dále ovlivní. Nezáleží na velikosti znaků = **non-case-sensitive**. 

Tagy mohou mít **atributy** (parametry tagu zapisované uvnitř ostrých závorek). K atributům se přiřazujou hodnoty pomocí `=hodnota`, pokud hodnota je jednoslovná, může být napsána bez uvozovek. Pokud však obsahuje mezeru, uvozovky být musí.

`<p color=blue> == <p color="blue">`

Pravidla mezer:
- 2 a víc mezer jsou počítána jako jedna mezera
- konec řádku je chápán jako mezera
- tabulátor je chápán jako mezera

### Párové tagy
Mají **začátek** a **konec**, s obsahem mezi nimi. Syntaxe: `<tag>` + obsah +  `</tag>`.
**Tag nesmí mít koncovou značku před tím, než ji budou mít všechny vnořené tagy.**


Ukázka:
```HTML
<p>Toto je odstavec</p>
```
Příklady:
- `<html>`, `<head>`, `<body>`, `<div>`, `<section>`, `<p>`, `<h1>–<h6>`, `<a>`, `<span>`, `<ul>`, `<li>`, ...

### Nepárové tagy
Nepárové tagy nemají ukončovací značku, **neuzavírají žádný text**. Syntaxe: `<tag>`.

Ukázka:
```HTML
<img src="cat.png">
```
Příklady:
- `<br>`, `<hr>`, `<img>`, `<input>`, `<meta>`, `<link>`, `<source>`, ...

### Přehled základních tagů
- `h1` - hlavní nadpis
- `h2` - podnadpis
- `h3-h6` - podnadpisy další úrovně
- `p` - odstavec
- `strong` - tučné písmo (**důležitá část textu** - pro SEO a čtečky obrazovky)
	- `b` - tučné písmo (pouze vizuálně)
- `em` - kurzíva **(zdůrazněná část textu**, fráze - taky pro čtečky obrazovky)
	- `i` - kurzíva (pouze vizuálně)
- `br` - zalomení řádku
- `a` - odkaz
- `img` - obrázek
- `ol` - číslovaný seznam
- `ul` - seznam s odrážkami
	- `li` - položka seznamu (číslovaného či odrážkovaného)
- `table` - tabulka
	- `tr` - řádek tabulky
	- `td` - buňka tabulky
	- `th` - buňka záhlaví tabulky
	- `caption` - nadpis tabulky
	- `thead` - záhlaví tabulky
	- `tbody` - tělo tabulky
	- `tfoot` - patička tabulky
- `div` - blok

### Rozvržení webové stránky
- `header` - záhlaví webové stránky, záhlaví článku
- `nav` - navigační menu
- `main` - obsah stránky, blok mezi záhlavím a zápatím
- `article` - článek
- `section` - část webové stránky obsahující články určitého zaměření, nebo části článku
- `aside` - postranní panel obsahojící informace okrajově související s hlavní částí
- `footer` - zápatí webové stránky, zápatí článku

### Entity
Entity jsou speciální znaky zapisované pomocí kódu. Používají se pro znaky, které mají v HTML speciální význam.

Příklady:
- `&lt;   <`
- `&gt;   >`
- `&nbsp;   pevná mezera`
- `&copy;   ©`

## Základní struktura HTML dokumentu
Na začátku stránky je uvedena **verze jazyka (typ dokumentu - `DOCTYPE`)**. Vlastní html dokument (značka html) je rozdělen na **dvě části**:
- Hlavička (`head`) obsahuje informace pro internetový prohlížeč (metadata),
- tělo (`body`) tvoří vlastní obsah webové stránky.
```html
<!DOCTYPE html>  
<html>
	<head>
		<meta  charset="UTF-8">
		<title>Název stránky</title>
	</head>
	<body>
		Obsah stránky  
	</body>
</html>
```
Od verze HTML5 se již neuvádí číslo verze, zápis typu dokumentu se zjednodušil na prostý zápis: `<!DOCTYPE HTML>`.

### Hlavička
Obsahuje důležité informace pro prohlížeč.
- kódování češtiny - pro správné zobrazení písmen je potřeba uvést znakovou sadu, ve které jsme dokument vytvořili.
	- `<meta charset="UTF-8">`
	- (UTF-8 je univerzální znaková sada obsahující písmena všech světových jazyků a je součástí všech operačních systémů.)
- titulek - zobrazuje se v titulkovém pruhu internetového prohlížeče
	- `<title>Název stránky</title>`
- připojení stylů - propojení s externím CSS souborem
	- `<link rel="stylesheet" href="style.css">`
