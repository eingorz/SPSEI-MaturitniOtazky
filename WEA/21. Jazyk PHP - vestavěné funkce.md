# PHP
PHP je skriptovací jazyk běžící na serveru, který umožňuje dynamicky generovat obsah webových stránek.

Použití:
- generování HTML
- práce s databází
- zpracování formulářů
- práce se soubory
- implementace autentizace a session

## Základní syntaxe
```php
<?php
// výpis textu
echo "Ahoj světe!";

// proměnné
$cislo = 10;          // číselná hodnota
$text = "hello";      // textový řetězec
$pravda = true;       // boolean

// pole
$pole = [1, 2, 3];    // indexované pole
$osoba = ["jmeno" => "Jan", "vek" => 25]; // asociativní pole
?>
```
Komentáře:
- `//` – jeden řádek
- `/* … */` – víceřádkový komentář

## Vestavěné funkce
**echo**
- vypisuje text nebo hodnoty na stránku.
    ```php
    echo "Ahoj světe!";
    ```

**include**
- vloží obsah souboru, pokud existuje.
    ```php
    include "header.php";  // pokud soubor chybí, zobrazí upozornění
    ```

**require**
- Povinné vložení souboru – pokud soubor chybí, PHP vyhodí fatální chybu.
    ```php
    require "config.php";
    ```

**isset**
- Kontrola, zda proměnná existuje.
    ```php
    if (isset($jmeno)) {
    echo "Proměnná je nastavena";
    }
    ```

**empty**
- Kontrola, zda proměnná je prázdná.
    ```php
    if (empty($heslo)) {
    echo "Pole heslo je prázdné";
    }
    ```

## Funkce pro pole
```php
$pole = [3, 1, 4, 2];

// počet prvků v poli
echo count($pole); // 4

// přidání prvku na konec
array_push($pole, 5); // [3,1,4,2,5]

// odstranění posledního prvku
array_pop($pole); // odstraní 5

// přidání prvku na začátek
array_unshift($pole, 0); // [0,3,1,4,2]

// odstranění prvního prvku
array_shift($pole); // odstraní 0

// seřazení vzestupně
sort($pole); // [1,2,3,4]

// seřazení sestupně
rsort($pole); // [4,3,2,1]

// vyhledání hodnoty v poli
if (in_array(3, $pole)) {
    echo "3 je v poli";
}

// spojení polí
$pole2 = [5,6];
$nova_pole = array_merge($pole, $pole2); // [4,3,2,1,5,6]

// převod na řetězec
$str = implode(",", $pole); // "4,3,2,1"

// převod z řetězce na pole
$arr = explode(",", $str); // [4,3,2,1]
```
## Funkce pro text
**Pro práci s multibyte znaky (např. čeština) se má používat mb_strlen(), mb_substr().**
```php
// základní řetězec
$text = "Hello PHP World!";

// délka textu
echo strlen($text); // 16

// výřez z textu
echo substr($text, 6, 3); // "PHP"

// nahrazení části textu
echo str_replace("World", "Universe", $text); // "Hello PHP Universe!"

// převod na velká/malá písmena
echo strtoupper($text); // "HELLO PHP WORLD!"
echo strtolower($text); // "hello php world!"

// odstranění mezer na začátku a konci
$text2 = "  test  ";
echo trim($text2); // "test"

// hledání podřetězce
$pozice = strpos($text, "PHP"); // 6

// rozdělení textu na pole
$slova = explode(" ", $text); // ["Hello", "PHP", "World!"]

// spojení pole na řetězec
$novy_text = implode("-", $slova); // "Hello-PHP-World!"
```
## Funkce pro datum
```php
// aktuální datum a čas
echo date("Y-m-d H:i:s"); // např. 2026-03-31 14:25:00

// pouze datum
echo date("d.m.Y"); // 31.03.2026

// pouze čas
echo date("H:i:s"); // 14:25:00

// aktuální timestamp (sekundy od 1.1.1970)
$ts = time();
echo $ts;

// převod čitelného datumu na timestamp
$ts2 = strtotime("2026-04-01 12:00:00");

// přidání nebo odečtení času
$zitra = date("Y-m-d", strtotime("+1 day")); // zítřek
$predtym = date("Y-m-d", strtotime("-7 days")); // před 7 dny

// rozdíl mezi dvěma daty
$start = strtotime("2026-03-01");
$end = strtotime("2026-03-31");
$rozdil = ($end - $start)/86400; // počet dní mezi daty
echo $rozdil; // 30
```
