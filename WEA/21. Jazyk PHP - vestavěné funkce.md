# PHP
PHP je skriptovací jazyk běžící na serveru.
Použití:
- generování HTML
- práce s databází
- zpracování formulářů

## Základní syntaxe
```php
<?php  
echo "Ahoj";  
?>
```
Proměnné:
```php
$cislo = 10;  
$text = "hello";
```
## Vestavěné funkce
**echo**
- vypisuje text

**include**
- vloží soubor

**require**
- povinné vložení

**isset**
- kontrola existence

**empty**
- kontrola prázdné hodnoty

## Funkce pro pole
```php
count()
array_push()
sort()
```
## Funkce pro text
```php
strlen()
substr()
str_replace()
```
## Funkce pro datum
```php
date()
time()
strtotime()
```
