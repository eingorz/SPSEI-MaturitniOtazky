# MVC
MVC znamená:
- Model
- View
- Controller

Odděluje logiku aplikace.
## Model
Pracuje s daty.
- databáze
- výpočty
## View
Zobrazuje data uživateli.
- HTML
- šablony
## Controller
Řídí aplikaci.
- přijímá požadavky
- volá model
- předává data pro view
## Předávání dat
Controller:
- získá data z modelu
- předá je view

View:
- zobrazí data
