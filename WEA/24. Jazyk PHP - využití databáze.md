# Princip využití databáze
Webové aplikace ukládají a načítají data z databáze.

Díky databázi můžeme uchovávat například:
- uživatele a jejich přihlašovací údaje
- objednávky v e-shopu
- články nebo obsah webu

Databáze umožňuje efektivní a bezpečné ukládání dat, vyhledávání a jejich aktualizaci.

## Rozšíření PHP pro databáze
PHP nabízí několik způsobů, jak komunikovat s databázemi, zejména MySQLi a PDO.
### MySQLi
Rozšíření pro práci s MySQL databázemi. Podporuje procedurální i objektový přístup.
## Procedurální přístup
```php
$conn = mysqli_connect("localhost", "root", "", "db");

// nastavení kódování
mysqli_set_charset($conn, "utf8");

// provedení dotazu
$result = mysqli_query($conn, "SELECT * FROM uzivatele");

// načtení výsledku jako asociativní pole
while($row = mysqli_fetch_assoc($result)) {
    echo $row["login"];
}
```

### důležité funkce
```php
mysqli_connect() // připojení k databázi
mysqli_set_charset() // nastavení kódování
mysqli_query() // provedení SQL dotazu
mysqli_fetch_assoc() // načtení řádku výsledku jako asociativní pole
```

## Objektový přístup
```php
$conn = new mysqli("localhost", "root", "", "db");

// kontrola připojení
if ($conn->connect_error) {
    die("Chyba připojení: " . $conn->connect_error);
}

// provedení dotazu
$result = $conn->query("SELECT * FROM uzivatele");

// načtení výsledku
while($row = $result->fetch_assoc()) {
    echo $row["login"];
}
```
Hlavní metody a atributy:
- query() – provedení dotazu
- prepare() – příprava prepared statement
- fetch_assoc() – načtení výsledku jako asociativní pole
- connect_error – kontrola chyb připojení

## PDO (PHP Data Objects)
PDO je **univerzální rozhraní pro práci s databázemi**.
**Podporuje více databázových systémů** (MySQL, PostgreSQL, SQLite…) a **bezpečné prepared statements**, což zvyšuje **bezpečnost proti SQL injection**.

Výhody PDO:
- Podpora více databází
- Bezpečné prepared statements
- Flexibilní nastavení atributů (ATTR_ERRMODE, ATTR_DEFAULT_FETCH_MODE)
- Snadná práce s transakcemi

Příklad:
```php
try {
    $pdo = new PDO("mysql:host=localhost;dbname=db;charset=utf8", "root", "");
    $pdo->setAttribute(PDO::ATTR_ERRMODE, PDO::ERRMODE_EXCEPTION);

    // prepared statement
    $stmt = $pdo->prepare("SELECT * FROM uzivatele WHERE role = ?");
    $stmt->execute(["ucitel"]);

    while($row = $stmt->fetch(PDO::FETCH_ASSOC)) {
        echo $row["login"];
    }
} catch (PDOException $e) {
    echo "Chyba připojení: " . $e->getMessage();
}
```
