# Princip využití databáze
Webové aplikace ukládají data do databáze.
Například:
- uživatelé
- objednávky
- články

## Rozšíření PHP pro databáze

### MySQLi
rozšíření pro práci s MySQL.
## Procedurální přístup
```php
$conn = mysqli_connect("localhost","root","","db");
```
### důležité funkce
```php
mysqli_connect()
mysqli_set_charset()
mysqli_query()
mysqli_fetch_assoc()
```
## Objektový přístup
```php
$conn = new mysqli("localhost","root","","db");
```
Metody:
- query()
- prepare()
- fetch_assoc()

## PDO
PDO znamená **PHP Data Objects**.
Výhody:
- podporuje více databází
- bezpečné prepared statements

Příklad:
```php
$pdo = new PDO("mysql:host=localhost;dbname=db","root","");
```
