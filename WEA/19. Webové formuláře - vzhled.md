## Pseudotřídy
Pseudotřídy umožňují **stylovat prvky podle jejich stavu nebo pozice**.
Používají se hlavně u formulářů pro zpětnou vazbu.
### focus
Styluje prvek, který má aktivní kurzor (do kterého vpisuje).

Použití:
- zvýraznění aktivního pole
```css
input:focus {  
border: 2px  solid  blue;  
}
```
Aktivní pole ohraničí modrou plnou 2 pixely tlustou čárou.

### valid / invalid
Používají se u formulářů s validací.

Například pro:
- `required` = povinné pole 
- `pattern` = vstupní maska (co se může vložit - čísla, text, ... ; velikost - 4 čísla, 3 až 16 znaků, ...)
- `type="email"` = validní emailová adresa (obsahuje `@`)
```css
input:valid {
	border: green;
}
input:invalid {
	border: red;
}
```
Význam:
- uživatel vidí, zda zadal správná data

### required / optional
Styluje pole, která jsou povinná a nepovinná.
```css
input:required {
	background-color: #ff7272;
}

input:optional {
    background-color: #437df1;
}
```

### checked
Používá se pro:
- checkbox
- radio button
```css
input:checked {
	background-color: green;
}
```
### disabled
Styluje zakázaná pole.
```css
input:disabled {
background-color: gray;
}
```
## Atributové selektory
Vyberou prvek podle **hodnoty atributu**.
### podle typu input
```css
input[type="text"] {
border: 1px  solid  black;
}
```
### podle existence atributu
```css
input[required] {
background: yellow;
}
```
### podle hodnoty atributu
```css
a[target="_blank"]{...}
```
## Další selektory
### Přímý potomek
```css
div  >  p {...}
```
vybere pouze **přímé potomky**
### Následník
```css
h1 + p {...}
```
vybere **první odstavec za h1**
### Obecný sourozenec
```css
h1 ~ p {...}
```
vybere všechny odstavce za h1.
