## Pseudotřídy
Pseudotřídy umožňují **stylovat prvky podle jejich stavu**.
Používají se hlavně u formulářů.
### focus
Styluje prvek, který má aktivní kurzor.
```css
input:focus {  
border: 2px  solid  blue;  
}
```
Použití:
- zvýraznění aktivního pole

### valid / invalid
Používají se u formulářů s validací.
```css
input:valid {
	border: green;
}
input:invalid {
	border: red;
}
```
Význam:
- uživatel vidí, zda zadal správná data
### required
Styluje pole, která jsou povinná.
```css
input:required {
	background-color: #fff3cd;
}
```
### checked
Používá se pro:
- checkbox
- radio button
```css
input:checked {
	background-color: green;
}
```
### disabled
Styluje zakázaná pole.
```css
input:disabled {
background-color: gray;
}
```
## Atributové selektory
Vyberou prvek podle **hodnoty atributu**.
### podle typu input
```css
input[type="text"] {
border: 1px  solid  black;
}
```
### podle existence atributu
```css
input[required] {
background: yellow;
}
```
### podle hodnoty atributu
```css
a[target="_blank"]{...}
```
## Další selektory
### Přímý potomek
```css
div  >  p {...}
```
vybere pouze **přímé potomky**
### Následník
```css
h1 + p {...}
```
vybere **první odstavec za h1**
### Obecný sourozenec
```css
h1 ~ p {...}
```
vybere všechny odstavce za h1.
