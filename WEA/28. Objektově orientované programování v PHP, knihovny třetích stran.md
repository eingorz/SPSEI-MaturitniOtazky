# Objektově orientované programování v PHP

OOP je programovací paradigma založené na objektech.
Výhody:
- přehlednost
- znovupoužitelnost
- modularita
## Třída
```php
class Auto {  
	public $barva;  
}
```
## Objekt
```php
$auto = new Auto();
```
## Metody
```php
function jed() {  
	echo "Auto jede";  
}
```
## Viditelnost
- public
- private
- protected
## Dědičnost
```php
class ElektroAuto extends Auto
```
## Statické metody
```php
static function test()
```

Používají se pro ošetření chyb.
```php
try  
catch
```
