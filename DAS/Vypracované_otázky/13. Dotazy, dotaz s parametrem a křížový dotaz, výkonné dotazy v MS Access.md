# Dotazy
Dotazy (queries) v databázových systémech, jako je Microsoft Access, slouží k získávání, filtrování a úpravě dat uložených v tabulkách.

Pomocí dotazů můžeme:
- vybírat konkrétní data (**SELECT**)
- filtrovat podle podmínek (**WHERE**)
- spojovat více tabulek (**JOIN**)
- provádět výpočty (**SUM, AVG**)
- upravovat data (**UPDATE, DELETE**)

Nejčastější typ dotazu je výběrový dotaz (**SELECT**).
Ukázka základního dotazu:
```SQL
SELECT jmeno, prijmeni
FROM Studenti;
```
Tento dotaz vypíše jména a příjmení všech studentů z tabulky Studenti.

## Dotaz s podmínkou `WHERE`
Pomocí klauzule **WHERE** můžeme omezit výběr dat podle určité podmínky.

Ukázka:
```SQL
SELECT jmeno, prijmeni
FROM Studenti
WHERE vek > 18;
```
Tento dotaz vrátí pouze studenty starší než 18 let.

Další operátory:
- = (rovno)
- <> (nerovno)
- (> <> = <=)
- **LIKE** (vyhledávání podle vzoru):
    ```SQL
    SELECT *
    FROM Studenti 
    WHERE prijmeni LIKE "Novák*";
    ```
    V Microsoft Access se místo % používá *.

## Dotaz s parametrem
Parametrický dotaz umožňuje uživateli **zadat hodnotu až při spuštění dotazu**.
V MS Access se parametr zapisuje **do hranatých závorek [ ]**.

Ukázka:
```SQL
SELECT jmeno, prijmeni
FROM Studenti
WHERE vek > [Zadej minimální věk:];
```
Po spuštění se zobrazí dialog (Zadej minimální věk:), kde uživatel zadá hodnotu.

Výhody:
- dotaz je univerzální (nemusíme ho přepisovat),
- zvyšuje interaktivitu,
- vhodné pro formuláře.

Další příklad:
```SQL
SELECT *
FROM Objednavky
WHERE datum = [Zadej datum:];
```

## Křížový dotaz (Crosstab Query)
Křížový dotaz slouží k souhrnnému zobrazení dat – podobně jako kontingenční tabulka v Excelu.

### Křížový dotaz - SQL
Používá se klauzule TRANSFORM ... PIVOT.

Ukázka:
```SQL
TRANSFORM SUM(cena) AS Celkem
SELECT zakaznik
FROM Objednavky
GROUP BY zakaznik
PIVOT rok;
```
Co dělá:
- řádky = zákazníci
- sloupce = roky
- hodnoty = součet ceny objednávek
- Vysvětlení:
- TRANSFORM = určuje agregaci
- SELECT = řádky
- PIVOT = sloupce

### Křížový dotaz - Návrhové zobrazení
Vybírá se z tabulek. 
Používá se:
- **Záhlaví řádku** - 1 záznam (např. žák)
- **Záhlaví sloupce** - sloupec hodnoty (např. předmět)
- **Hodnota** - musí používat agregační funkci (např. průměr známek z předmětu)

Křížové dotazy jsou velmi užitečné pro:
- statistiky
- přehledy
- reporty

## Výkonné dotazy (Action Queries)
Výkonné dotazy slouží ke **změně dat v databázi**. Tyto dotazy mění data **trvale**!

Typy výkonných dotazů:
1. **UPDATE** (aktualizační dotaz)
    - Změní existující data.
    ```SQL
    UPDATE Studenti
    SET vek = vek + 1;
    ```
    Zvýší věk všech studentů o 1.

2. **DELETE** (odstranění záznamů)
    - Smaže data
    ```SQL
    DELETE FROM Studenti
    WHERE vek < 18;
    ```
    Smaže studenty mladší než 18 let.

3. **INSERT** (přidání dat)
    - Přidá nový záznam
    ```SQL
    INSERT INTO Studenti (jmeno, prijmeni, vek)
    VALUES ("Jan", "Novák", 20);
    ```
    Přidá do tabulky Studenti Jan Novák 20

4. **CREATE TABLE** (vytvoření tabulky)
    ```SQL
    CREATE TABLE Test (
    id AUTOINCREMENT,
    jmeno TEXT
    );
    ```
5. **MAKE TABLE** (vytvoření tabulky z dotazu)
    ```SQL
    SELECT *
    INTO NovaTabulka
    FROM Studenti
    WHERE vek > 18;
    ```
