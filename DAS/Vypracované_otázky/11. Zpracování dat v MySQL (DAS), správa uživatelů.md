## 11.1 Popis importovaného souboru
- Importovaný soubor obsahuje data, která se nahrávají do databáze.

- Nejčastější formáty:
	- **.sql** – obsahuje SQL příkazy (CREATE, INSERT…)
	- **.csv** – data oddělená oddělovačem (čárka, středník, tabulátor)
	- **.txt** – textový soubor se strukturou podobnou CSV
	- **.xml / JSON** – strukturovaná data (méně časté)

## 11.3 Správa uživatelů v MySQL
### 11.3.1 Vytvoření uživatele
```mysql
CREATE USER [IF NOT EXISTS] 'jmeno_uzivatele'@'adresa' [IDENTIFIED BY 'heslo'];
```
- `adresa` - určení odkud se může uživatel připojit, použíjeme `%` pokud chceme povolit připojení odkudkoliv

### 11.3.2 Přidělení oprávnění
```mysql
GRANT prava ON kde TO 'jmeno_uzivatele'@'adresa' [WITH GRANT OPTION];
```
- `prava` - ALL PRIVILEGES, SELECT, INSERT, UPDATE, DELETE
- `kde` - pro jakou tabulku/databázi `database.*`
- `WITH GRANT OPTION`- dá uživateli možnost nastavit jeho práva a rozsah dalším uživatelům, nastavená práva zaniknou pokud je uživateli GRANT OPTION odebrána

### 11.3.3 Odebrání oprávnění
```mysql
REVOKE INSERT, GRANT OPTION ON databaze.student FROM 'uzivatel'@'localhost';

REVOKE ALL [PRIVILEGES] ON databaze.student FROM 'uzivatel'@'localhost';
```

### 11.3.4 Zobrazení oprávnění
```mysql
SHOW GRANTS [FOR 'uzivatel'@'localhost'];
```
- pokud nespecifikujeme `FOR` - zobrazí se práva pro aktuálního uživatele

### 11.3.5 Smazání uživatele
```mysql
DROP USER 'uzivatel'@'localhost';
```

### 11.3.6 Změna hesla
```mysql
SET PASSWORD [FOR 'jeffrey'@'localhost'] = 'auth_string';
```
- pokud nespecifikujeme `FOR` - změní se heslo pro aktuálního uživatele

### 11.3.7 Typy oprávnění
- SELECT – čtení dat
- INSERT – vkládání dat
- UPDATE – úprava dat
- DELETE – mazání dat
- ALL PRIVILEGES – všechna práva
