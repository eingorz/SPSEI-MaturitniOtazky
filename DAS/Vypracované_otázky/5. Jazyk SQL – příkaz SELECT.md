## 5.1 Povinné a nepovinné části příkazu SELECT
```MySQL
SELECT sloupce  
FROM tabulka  
[WHERE podmínka]  
[GROUP BY sloupce]  
[HAVING podmínka]  
[ORDER BY sloupce];
```

### 5.1.1 Povinné části:
- **SELECT** – určuje, které sloupce chceme zobrazit
- **FROM** – určuje tabulku (nebo tabulky), ze kterých data bereme

### 5.1.2 Nepovinné části:
- **WHERE** – filtruje řádky (podmínka výběru)
- **GROUP BY** – seskupuje řádky do skupin
- **HAVING** – filtruje skupiny (na rozdíl od WHERE)
- **ORDER BY** – řazení výsledků
- **DISTINCT** – odstranění duplicit
- **JOIN** – spojování tabulek

## 5.2 Výběr zobrazených sloupců
### 5.2.1 Výběr konkrétních sloupců:
```Mysql
SELECT jmeno, prijmeni FROM zakaznici;
```

### 5.2.2 Výběr všech sloupců:
```Mysql
SELECT * FROM zakaznici;
```

### 5.2.3 Přejmenování sloupců (alias):
```Mysql
SELECT jmeno AS jméno_zákazníka
FROM zakaznici;
```

### 5.2.4 Filtrování řádků (WHERE)
```Mysql
SELECT * FROM zakaznici
WHERE mesto = 'Ostrava';
```

### 5.2.5. Omezení duplicit (DISTINCT):
```Mysql
SELECT DISTINCT mesto FROM zakaznici;
```

## 5.3 Seskupování záznamů (GROUP BY)
### 5.3.1 Proč se používá:
- pro **souhrnné informace** (např. kolik zákazníků je v každém městě)
- umožňuje použití **agregačních funkcí**
```Mysql
SELECT mesto, COUNT(*)   
FROM zakaznici  
GROUP BY mesto;
```

### 5.3.2 Agregační funkce:
- **COUNT()** – počet záznamů
- **SUM()** – součet
- **AVG()** – průměr
- **MIN()** – minimum
- **MAX()** – maximum

### 5.3.3 HAVING (filtr skupin):
```Mysql
SELECT mesto, COUNT(*)   
FROM zakaznici  
GROUP BY mesto  
HAVING COUNT(*) > 5;
```

## 5.4 Pohledy (VIEW) v databázích
- Je virtuální tabulka vytvořená na základě SELECT dotazu.
- Neobsahuje vlastní data, pouze uložený dotaz.

### 5.4.1 K čemu slouží:
- zjednodušení složitých dotazů
- zvýšení bezpečnosti (omezíme přístup jen na část dat)
- znovupoužitelnost
- abstrakce (uživatel nevidí strukturu databáze)

## 5.5 Vytvoření a úprava pohledu
### 5.5.1 Vytvoření pohledu
```Mysql
CREATE VIEW aktivni_zakaznici AS  
SELECT jmeno, prijmeni, mesto  
FROM zakaznici  
WHERE aktivni = 1;
```
### 5.5.2 Použití pohledu
```Mysql
SELECT * FROM aktivni_zakaznici;
```

### 5.5.3 Úprava pohledu
```Mysql
CREATE OR REPLACE VIEW aktivni_zakaznici AS
SELECT jmeno, prijmeni
FROM zakaznici
WHERE aktivni = 1;
```

### 5.5.4 Smazání pohledu
```Mysql
DROP VIEW aktivni_zakaznici;
```

## 5.6 Shrnutí
- `SELECT` slouží k výběru dat z databáze
- Povinné části: **SELECT, FROM**
- Důležité klauzule: **WHERE, GROUP BY, HAVING, ORDER BY**
- Umožňuje:
    - výpočty
    - filtrování
    - seskupování
    - odstranění duplicit
- **Pohled (VIEW)** = uložený SELECT dotaz bez vlastních dat
