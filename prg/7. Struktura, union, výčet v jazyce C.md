- Heterogenní strukturované datové typy v jazyce C + typedef
# Struct
- Jedná se o datový typ jenž umožňuje programátorovi deklarovat a pracovat se skupinou proměnných v operační pamětí jako s celkem
- Struktura může obsahovat libovolné množství existujících, jednoduchých i strukturovaných datových typů
## Deklarace
``` C
struct mujStruct {
	int cislo;
	char pismenko;
};

void main() {
	struct mujStruct s1;
}
```
## Přistupování ke structu
```C
void main(){
	struct mujStruct s1;
	s1.cislo = 1;
	s1.pismenko = 'c';
	
    printf("%d\n", s1.cislo);  
    printf("%c\n", s1.pismenko);
}
```
## Incializace celého structu
```C
void main(){
	struct mujStruct s1 = {4, 'N'};
}
```
- *Poznámka: Při incializaci lze nastavit string "klasicky", avšak pokud mu budeme chtít později změnit hodnotu, je nutno použít např. funkci strcpy(), či popořadě procházet prvky*
## Kopírování structu
```C
struct mujStruct s1 = {4, 'N'};
struct mujStruct s2;

s2 = s1;
```
## Dynamická alokace structu
```C
#include <stdio.h> 
#include <stdlib.h> 
#include <string.h>

struct Zamestnanec {
    int id;
    char *jmeno; // Ukazatel pro dynamicky alokovaný řetězec
};

int main() {
    // 1. DYNAMICKÁ ALOKACE SAMOTNÉ STRUKTURY
    struct Zamestnanec *z1 = (struct Zamestnanec *)malloc(sizeof(struct Zamestnanec));
    
    // Vždy kontrolujeme, zda alokace proběhla úspěšně
    if (z1 == NULL) {
        printf("Chyba: Nepodarilo se alokovat pamet pro strukturu!\n");
        return 1;
    }

    // 2. ZÁPIS DAT
    // Protože z1 je UKAZATEL, nepoužíváme tečku (.), ale šipku (->)
    z1->id = 42;

    // 3. DYNAMICKÁ ALOKACE DAT UVNITŘ STRUKTURY
    // Alokujeme místo pro 50 znaků textu
    z1->jmeno = (char *)malloc(50 * sizeof(char));
    
    if (z1->jmeno != NULL) {
        strcpy(z1->jmeno, "Karel Novak"); // Nakopírujeme text do alokované paměti
    } else {
        printf("Chyba: Nepodarilo se alokovat pamet pro jmeno!\n");
        free(z1);
        return 1;
    }
    printf("Zamestnanec ID: %d, Jmeno: %s\n", z1->id, z1->jmeno);


    // 4. UVOLNĚNÍ PAMĚTI

    free(z1->jmeno); 
    free(z1);        
    z1 = NULL;

    return 0;
}

```

## Využití s typedef
- Pomocí klíčového slova `typedef` můžeme vytvořit alias pro naši strukturu, abychom při deklaraci proměnných nemuseli neustále psát slovo `struct`.
```C
typedef struct {
	int x;
	int y;
} Bod;

void main() {
	// Místo "struct Bod b1;" píšeme pouze:
	Bod b1 = {10, 20};
	printf("Souradnice: %d, %d\n", b1.x, b1.y);
}
```

# Union
- Jedná se o strukturovaný datový typ velmi podobný struktuře (`struct`), ale s jedním zásadním rozdílem: **všechny jeho položky sdílejí stejné místo v paměti**.
- Velikost unionu v paměti je dána velikostí jeho největší (datově nejnáročnější) položky.
- V jeden okamžik může union smysluplně uchovávat hodnotu pouze jedné ze svých položek (zápisem do jedné položky se přepíše/znehodnotí hodnota ostatních).

## Deklarace
```C
union mujUnion {
	int cislo;
	char pismenko;
};

void main() {
	union mujUnion u1;
}
```

## Přistupování k unionu
- Přistupuje se stejně jako u struktury pomocí tečkové notace (`.`), ale musíme si dávat pozor na přepisování paměti.
```C
void main(){
	union mujUnion u1;
	
	u1.cislo = 65;
	printf("%d\n", u1.cislo); // Vypíše 65
	
	// Nyní zapíšeme do jiné položky - tím se přepíše hodnota v paměti!
	u1.pismenko = 'A';
	printf("%c\n", u1.pismenko); // Vypíše 'A'
	
	// Kdybychom teď vypsali u1.cislo, dostaneme 65 (protože 'A' má v ASCII hodnotu 65)
}
```

## Inicializace unionu
- Při inicializaci pomocí složených závorek se inicializuje **pouze první položka** deklarovaná v unionu.
```C
void main(){
	union mujUnion u1 = {42}; // Inicializuje pouze položku 'cislo'
}
```

## Kopírování unionu
- Funguje naprosto stejně jako u struktur.
```C
union mujUnion u1 = {42};
union mujUnion u2;

u2 = u1; // Zkopíruje celý obsah paměti unionu
```

## Dynamická alokace unionu
- Opět stejný princip jako u struktur, k prvkům přistupujeme přes operátor šipky (`->`).
```C
#include <stdio.h> 
#include <stdlib.h> 

union Data {
    int i;
    float f;
};

int main() {
    // 1. DYNAMICKÁ ALOKACE
    union Data *u1 = (union Data *)malloc(sizeof(union Data));
    
    if (u1 == NULL) {
        printf("Chyba alokace!\n");
        return 1;
    }

    // 2. ZÁPIS A ČTENÍ DAT
    u1->i = 10;
    printf("Cislo: %d\n", u1->i);

    // Zápis do floatu přepíše původní int
    u1->f = 3.14;
    printf("Float: %f\n", u1->f);

    // 3. UVOLNĚNÍ PAMĚTI
    free(u1);
    u1 = NULL;

    return 0;
}
```

## Využití s typedef
- Stejně jako u struktur můžeme i u unionů použít `typedef` pro zjednodušení zápisu.
```C
typedef union {
	int celeCislo;
	float desetinneCislo;
} Cislo;

void main() {
	// Místo "union Cislo c1;" píšeme pouze:
	Cislo c1;
	c1.celeCislo = 42;
	printf("Cele cislo: %d\n", c1.celeCislo);
}
```

# Enum (Výčtový typ)
- Jedná se o uživatelsky definovaný datový typ, který se skládá z množiny pojmenovaných celočíselných konstant.
- Slouží především ke zpřehlednění a lepší čitelnosti kódu – místo "magických čísel" (např. 0, 1, 2) používáme srozumitelná slova (např. `PONDELI`, `UTERY`).
- Standardně má první prvek hodnotu `0` a každý další prvek má automaticky hodnotu o 1 vyšší.
- Hodnoty jednotlivých prvků lze ale při deklaraci libovolně změnit.

## Deklarace
```C
enum Dny {
	PONDELI,  // 0
	UTERY,    // 1
	STREDA,   // 2
	CTVRTEK,  // 3
	PATEK,    // 4
	SOBOTA,   // 5
	NEDELE    // 6
};

void main() {
	enum Dny dnes;
}
```

## Přistupování a použití
- S proměnnou typu `enum` pracujeme v podstatě jako s běžným celým číslem (`int`). Můžeme ji porovnávat i vypisovat.
```C
void main(){
	enum Dny dnes = STREDA;
	
	if (dnes == STREDA) {
		printf("Je streda, jeji ciselna hodnota je: %d\n", dnes); // Vypíše 2
	}
}
```

## Inicializace s vlastními hodnotami
- Číselné hodnoty můžeme explicitně nastavit. Prvky bez definované hodnoty pak automaticky navazují na předchozí prvek (+1).
```C
enum Stavy {
	CHYBA = -1,
	CEKANI = 0,
	BEZI,        // Bude mít automaticky hodnotu 1
	DOKONCENO = 5
};
```

## Využití s typedef
- Klíčové slovo `typedef` se v jazyce C používá k vytvoření aliasu pro datový typ (funguje to i pro `struct` a `union`).
- Zjednodušuje to následnou deklaraci proměnných, protože pak už nemusíme neustále psát klíčové slovo `enum`.
```C
typedef enum {
	NEPRAVDA, // 0
	PRAVDA    // 1
} Boolean;

void main() {
	// Místo "enum Boolean stav;" píšeme už jen zkrácený alias:
	Boolean stav = PRAVDA;
	
	if (stav == PRAVDA) {
		printf("Pravda!\n");
	}
}
```
