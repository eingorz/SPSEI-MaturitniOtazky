
### Preprocesor jako krok v procesu zpracování programu v jazyce C

Než se program v jazyce **C** přeloží do spustitelného souboru, prochází několika kroky:

1. **Preprocessing (předzpracování)** – zpracování direktiv začínajících `#`
2. **Compilation (kompilace)** – překlad do objektového kódu
3. **Linking (linkování)** – spojení objektových souborů a knihoven do výsledného programu

**Preprocesor** tedy pracuje **ještě před samotnou kompilací**.

---
# Preprocesor v jazyce C

### Co je preprocesor

**Preprocesor** je nástroj, který zpracovává speciální direktivy, které začínají znakem `#`. Tyto direktivy nejsou součástí samotného jazyka C, ale řídí práci preprocesoru.

**Preprocesor** neprovádí kontrolu syntaxe C ani sémantickou analýzu kódu. Pouze provádí textové náhrady a vkládání souborů
### Vstup preprocesoru

Vstupem je:
- **zdrojový kód programu v jazyce C**
- typicky soubor: `program.c`

Preprocesor:
- rozšíří makra
- vloží hlavičkové soubory
- odstraní komentáře
- vyhodnotí podmíněný překlad

Výstupem je **upravený zdrojový kód**, který jde do kompilátoru. Preprocesor produkuje upravený zdrojový kód (často nazývaný "translation unit" nebo "preprocessed source file").

---
# Direktivy preprocesoru

### Co jsou direktivy

**Direktivy preprocesoru** jsou speciální instrukce:
- začínají znakem `#`
- nejsou to příkazy jazyka C
- zpracovává je **preprocesor**

Například:

```c
#include <stdio.h>  
#define PI 3.14  
#ifdef DEBUG
```

---
## Základní direktivy

### `#include`

Slouží pro **vložení souboru** do programu.

```c
#include <stdio.h>  
#include "moje_funkce.h"
```

Dva způsoby:
- `<soubor>` → systémové knihovny
- `"soubor"` → lokální soubory

---
### `#define`

Slouží pro **definici makra**.
Makro je **textová náhrada**, kterou preprocesor dosadí do programu.

---
# Makra
## Jednoduchá makra (bez parametrů)

Používají se jako **symbolické konstanty**.
```c
#define PI 3.14159  
#define MAX 100
```

Použití:
```c
double obsah = PI * r * r;
```
Po preprocessingu:
```c
double obsah = 3.14159 * r * r;
```

---
## Makra s parametry

Makra mohou fungovat podobně jako funkce. Hodnota se zde ovšem nepodává za běhu programu, preprocesor opět pouze nahrazuje text textem.

```c
#define SQUARE(x) ((x) * (x))
```

Použití:
```c
int a = SQUARE(5);
```

Po preprocessingu:
```c
int a = ((5) * (5));
```

Většinou se v definici makra s parametrem parametr obaluje do kulatých závorek při použití, preprocesor totiž neřeší prioritu operátorů

Příklad:
```c
SQUARE(1+2)
```

Po rozšíření:
```c
1+2 * 1+2
```

nejdříve se vyhodnotí `2 * 1` uprostřed = špatný výsledek
Proto se používají **závorky**.

# Podmíněný překlad
Podmíněný překlad umožňuje **zapnout nebo vypnout části kódu** podle podmínek.

Používá se například pro:
- ladění programu
- různé verze programu
- různé platformy

---
## Direktiva `#ifdef`

Kód se zahrne **jen pokud je makro definováno**.

```c
#define DEBUG  
  
#ifdef DEBUG  
printf("Debug informace\n");  
#endif
```

---
## Direktiva `#ifndef`

Kód se přeloží **jen pokud makro není definováno**.

```c
#ifndef MAX  
#define MAX 100  
#endif
```

Často se používá v **hlavičkových souborech**.

---

## Direktiva `#if`

Podmínka může být i výraz.

```c
#define VERZE 2  
  
#if VERZE == 2  
printf("Verze 2 programu\n");  
#endif
```

---

## Direktivy `#else` a `#elif`

Používají se podobně jako v jazyce C.

```c
#define VERZE 1  
  
#if VERZE == 1  
printf("Verze 1\n");  
#elif VERZE == 2  
printf("Verze 2\n");  
#else  
printf("Neznámá verze\n");  
#endif
```

## Makro operátor `##` (token pasting)

Operátor **`##`** se používá v makrech preprocesoru pro **spojení dvou tokenů (částí kódu) do jednoho**.

- nazývá se **token pasting operator**
- používá se **pouze v makrech**
- provádí **textové spojení identifikátorů**

Preprocesor tak může **dynamicky vytvářet názvy proměnných nebo funkcí**.

---

### Základní princip

Syntaxe:
```c
#define MACRO(a, b) a ## b
```

Operátor `##` spojí `a` a `b` do jednoho tokenu.

---
### Praktický příklad – generování proměnných
```c
#define VAR(num) var##num  
  
int VAR(1) = 10;  
int VAR(2) = 20;
```

Po preprocessingu:
```c
int var1 = 10;  
int var2 = 20;
```

### K čemu se používá

Operátor `##` se používá například pro:
- **generování názvů proměnných**
- **generování funkcí**
- **makra pro knihovny**
- **automatické vytváření identifikátorů**

Často se používá ve **velkých knihovnách nebo frameworkách v C**.
