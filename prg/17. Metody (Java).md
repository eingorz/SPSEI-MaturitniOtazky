# 12. Metody v Javě (maturitní otázka)

## Co je metoda?

Metoda je pojmenovaný blok kódu, který vykonává určitou činnost. Používá se pro:

- znovupoužitelnost kódu
- přehlednost programu
- rozdělení problému na menší části

### Základní syntaxe metody
```Java
modifikátor návratový_typ název(parametry) {
    // tělo metody
    return hodnota; // pokud není void
}
```

Příklad:
```Java
public int secti(int a, int b) {
    return a + b;
}
```

### Základní pojmy

#### 1. Návratový typ (return type)

Určuje, jaký typ hodnota metoda vrací
např. `int`, `double`, `String`

Pokud nic nevrací → void

```Java
public void pozdrav() {
    System.out.println("Ahoj");
}
```

#### 2. Parametry (argumenty)

Vstupní hodnoty metody
mohou být:

- primitivní (`int`, `double`)
- referenční (`String`, objekty`)

```Java
public void vypisJmeno(String jmeno) {
    System.out.println(jmeno);
}
```

#### 3. Volání metody
```Java
int vysledek = secti(5, 3);
pozdrav();
```

### Modifikátory přístupu
| Modifikátor | Přístup              |
| ----------- | -------------------- |
| `public`    | odkudkoliv           |
| `private`   | pouze ve třídě       |
| `protected` | ve třídě a potomcích |
| (žádný)     | v rámci balíčku      |

## Statické vs. instanční metody

### Statická metoda (static)

- Patří třídě, ne objektu
- Volá se bez vytvoření instance

```Java
public static void vypis() {
    System.out.println("Statická metoda");
}
```

Volání:
```Java
MojeTrida.vypis();
```

### Instanční metoda

Patří konkrétnímu objektu (vyžaduje objekt se kterým pracuje - `this`)
```Java
public void pozdrav() {
    System.out.println("Ahoj");
    this.field = 1;
}
```

Volání:
```Java
MojeTrida obj = new MojeTrida();
obj.pozdrav();
```

K `obj` přistupujeme pomocí klíčového slova `this`

## Přetěžování a přepisování metod

### Přetěžování (Overloading)

Stejné jméno, různé parametry

```Java
int soucet(int a, int b) { return a + b; }
double soucet(double a, double b) { return a + b; }
```

### Přepisování (Overriding)

Redefinice metody v potomkovi

```Java
@Override
void zvuk() {
    System.out.println("Haf");
}
```

## Rekurze

Metoda volá sama sebe

```Java
public int faktorial(int n) {
    if (n == 1) return 1;
    return n * faktorial(n - 1);
}
```

## Předávání parametrů

### Hodnotou (primitivní typy)

Kopie hodnoty

```Java
public void zvys(int x) {
    x = x + 1;
}
```

### Referencí (objekty)

Předává se odkaz na objekt

```Java
public void pridejDoSeznamu(ArrayList<String> seznam) {
    seznam.add("Ahoj");
}
```

## Klíčové slovo return

Ukončí tok průběhu metody

### Brzký návrat (Early return)

Slouží k předčasnému ukončení toku průběhu metody v případě že nebyla splněna podmínka

Příklad:
```Java
if (objekt == null) return;
```

### Negative-space programování

Praktika obracení podmínek za využití brzkého návratu.
Eliminuje vnořené podmínky

Bez užití "Negative-space" programování
```Java
if (podminka1) {
    // kod1
    if (podminka2) {
        //kod2
        if (podminka3) {
            //kod3
        }
    }
}
```

S užitím "Negative-space" programování
```Java
if (!podminka1) return;
// kod1

if (!podminka2) return;
// kod2

if (!podminka3) return;
// kod3
```
(Přehlednější)

## Navrácení hodnoty

Je výstupem metody
Datový typ vrácené hodnoty určuje návratový typ
Pokud návratový typ je `void`, metoda nic vracet nemusí

```Java
public int vratPet() {
    return 5;
}
```

Metoda může mít více parametrů, ale jen jeden návratový typ
Pokud metoda není void, musí vždy vrátit hodnotu

## Příklad metody
```Java
public class Kalkulacka {

    public static int secti(int a, int b) {
        return a + b;
    }

    public static double prumer(int a, int b) {
        return (a + b) / 2.0;
    }

    public static void main(String[] args) {
        int soucet = secti(10, 20);
        double avg = prumer(10, 20);

        System.out.println("Součet: " + soucet);
        System.out.println("Průměr: " + avg);
    }
}
```
