# Charakteristika Controls
Komponenty ze skupiny **Controls** v knihovně **JavaFX** slouží k tvorbě grafického uživatelského rozhraní (GUI). Jsou to *interaktivní prvky, pomocí kterých uživatel komunikuje s aplikací.*

## Základní vlastnosti:
- dědí ze třídy Control
- mají vlastnosti (properties):
    - text (setText())
    - velikost (setPrefWidth())
    - styl (setStyle())
- podporují události (events):
    - kliknutí myši
    - stisk klávesy
    - změna hodnoty

### Nejčastější události:
- setOnAction() – kliknutí
- setOnMouseClicked()
- setOnKeyPressed()

# Komponenty Button, CheckBox, RadioButton
## Button (tlačítko)
- Slouží k vyvolání akce po kliknutí.

Vlastnosti:
- text
- grafika (ikona)

Událost:
- setOnAction()

Ukázka:
```Java
Button btn = new Button("Klikni na mě");

btn.setOnAction(e -> {
    System.out.println("Tlačítko bylo zmáčknuto");
});
```
## CheckBox (zaškrtávací pole)
Umožňuje výběr **ANO/NE**. (true/false)

Vlastnosti:
- isSelected()

Ukázka:
```Java
CheckBox cb = new CheckBox("Souhlasím");

cb.setOnAction(e -> {
    System.out.println(cb.isSelected());
});
```

## RadioButton
- Výběr jedné možnosti ze skupiny
- Používá se s ToggleGroup

#### ToggleGroup
Slouží k seskupení **radio** tlačítek, jinak by mohly být vybrány všechny.

Příklad:
- 2 RadioButtony (Elektrotechnika|Informatika):
    - **Bez ToggleGroup** - Mohu vybrat Elektrotechnika i Informatika zároveň
    - **S ToggleGroup** - Mohu vybrat jen jednu možnost

Ukázka:
```Java
ToggleGroup group = new ToggleGroup();

RadioButton r1 = new RadioButton("Muž");
RadioButton r2 = new RadioButton("Žena");

r1.setToggleGroup(group);
r2.setToggleGroup(group);
```
## Label, TextField, TextArea
### Label
Zobrazuje text (needitovatelný)

Ukázka:
```Java
Label label = new Label("Zadej jméno:");
```
### TextField
Jednořádkové vstupní pole

Vlastnosti:
- getText()
- setText()

### Ukázka:
```Java
TextField tf = new TextField();

Button btn = new Button("Vypiš");

btn.setOnAction(e -> {
    System.out.println(tf.getText());
});
```

### TextArea
Víceřádkový text

Ukázka:
```Java
TextArea ta = new TextArea();
ta.setPrefRowCount(5);
```

### ListView, ComboBox
#### ListView
Zobrazuje seznam položek

Vlastnosti:
- getItems()
- getSelectionModel()

Ukázka:
```Java
ListView<String> list = new ListView<>();

list.getItems().addAll("Java", "Python", "C++");

list.setOnMouseClicked(e -> {
    System.out.println(list.getSelectionModel().getSelectedItem());
});
```
#### ComboBox
Rozbalovací seznam

Ukázka:
```Java
ComboBox<String> combo = new ComboBox<>();

combo.getItems().addAll("Červená", "Modrá", "Zelená");

combo.setOnAction(e -> {
    System.out.println(combo.getValue());
});
```

### ColorPicker
Slouží k výběru barvy

Vlastnosti:
- getValue() → vrací barvu

Ukázka:
```Java
ColorPicker cp = new ColorPicker();

cp.setOnAction(e -> {
    System.out.println(cp.getValue());
});
```
