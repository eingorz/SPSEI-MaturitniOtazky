# 19. Znaky a řetězce (Java)

## 1. Znak (`char`)
- Datový typ pro **jeden znak**
- Velikost: **16 bitů (2 bajty)** – používá Unicode
- Zapisuje se do **apostrofů `' '`**

### Příklady:
```java
char a = 'A';
char b = '1';
char c = '#';
```
Unicode:

### Každý znak má své číselné označení
```Java
char znak = 65; // 'A'
Operace se znaky:
```
Lze porovnávat:
```Java
if (a == 'A') { ... }
```
Lze převádět:
```Java
char male = 'a';
char velke = Character.toUpperCase(male);
```
## 2. Řetězec (String)

Datový typ pro text (více znaků)

Není primitivní typ → je to objekt

Zapisuje se do uvozovek `" "`

Příklady:
```Java
String text = "Ahoj";
String prazdny = "";
```
## 3. Vlastnosti Stringu

Neměnitelný (immutable)
→ nelze změnit obsah, vždy vzniká nový objekt
```Java
String s = "Ahoj";
s = s + " světe"; // vytvoří nový String
```
## 4. Základní metody String
Délka:
```Java
text.length();
```
Přístup ke znaku:
```Java
text.charAt(0); // první znak
```
Porovnání:
```Java
text.equals("Ahoj");        // správně
text.equalsIgnoreCase("ahoj");
```

Nepoužívat:
```Java
text == "Ahoj"; // porovnává reference!
```
Podřetězec:
```Java
text.substring(1, 3);
```
Vyhledávání:
```Java
text.indexOf("o");
```
Nahrazení:
```Java
text.replace("Ahoj", "Čau");
```
Převody:
```Java
text.toUpperCase();
text.toLowerCase();
```
## 5. Spojování řetězců
```Java
String jmeno = "Jan";
String zprava = "Ahoj " + jmeno;
```
## 6. StringBuilder

Používá se pro efektivní úpravy textu

Na rozdíl od Stringu je měnitelný

Využívá kapacity, obdobně jako ArrayList charakterů.
Má kapacitu, při přidání znaků nad rámec kapacity se přealokuje, zkopíruje prvky a navýší kapacitu na dvojnásobek.

Příklad:
```Java
StringBuilder sb = new StringBuilder("Ahoj");
sb.append(" světe");
System.out.println(sb);
```
Výhoda:

Rychlejší při častých změnách

## Escape sekvence

Speciální znaky v řetězcích pro reprezentování nereprezentovatelných znaků
Začínají zpětným lomítkem
- `\n` - nový řádek (newline)
- `\r` - návrat jezdce (carriage return)
- `\t` - tabulátor
- `\b` - backspace (posunutí vzad, nikoliv oodstranění charakteru)
- `\"` - uvozovky (bez escape sekvence by ukončili string literál)
- `\\` - zpětné lomítko (bez escape sekvence by se považovalo za začátek escape sekvence)
