- **Funkce** je logický blok kódu, který jednou napíšeme a poté ho můžeme libovolně volat bez toho, abychom ho psali znovu a opakovali se. Dodržuje princip **DRY (Don't Repeat Yourself)**.
- Zvyšuje čitelnost kódu, usnadňuje ladění (debugging) a umožňuje rozdělení velkého problému na menší části (modularita).
- Taky se jim nadává podprogramy či subrutiny. Pokud funkce nevrací hodnotu (používá `void`), často se jí říká **procedura**.

## 1. Deklarace, definice a volání
- **Deklarace (prototyp):** Říká kompilátoru, jak se funkce jmenuje, jaký má návratový typ a jaké přijímá parametry. Píše se většinou nad `main()` nebo do hlavičkových souborů (`.h`).
- **Definice:** Obsahuje samotné tělo funkce (kód, který se provede).
- **Volání:** Spuštění funkce na konkrétním místě v programu.

```c
#include <stdio.h>

// Deklarace (prototyp) funkce
int secti(int a, int b);

int main() {
    int vysledek = secti(5, 3); // Volání funkce s argumenty 5 a 3
    printf("Soucet je: %d\n", vysledek);
    return 0;
}

// Definice funkce
int secti(int a, int b) {
    return a + b; // Návratová hodnota
}
```

## 2. Návratové typy a `return`
- Funkce může vracet jednu hodnotu pomocí klíčového slova `return`.
- Typ vracené hodnoty musí odpovídat datovému typu uvedenému před názvem funkce (`int`, `float`, `char`, atd.).
- Pokud funkce nic nevrací, používá se klíčové slovo **`void`**. V takovém případě `return;` pouze ukončí běh funkce (nebo se vůbec neuvádí).

## 3. Parametry a argumenty
- **Parametry:** Proměnné definované v hlavičce funkce (např. `int a, int b`).
- **Argumenty:** Konkrétní hodnoty předané funkci při její inicializaci / volání (např. `5, 3`).

### Způsoby předávání argumentů:
1. **Předávání hodnotou (Pass by Value):**
   - Funkce dostane pouze *kopii* hodnoty. Změna parametru uvnitř funkce neovlivní původní proměnnou.
2. **Předávání odkazem / ukazatelem (Pass by Reference / Pointer):**
   - Funkce dostane paměťovou adresu proměnné pomocí ukazatele (`*`). Změna hodnoty přes ukazatel změní původní proměnnou.

```c
#include <stdio.h>

void zmenHodnotu(int *cislo) {
    *cislo = 100; // Změní hodnotu na původní adrese
}

int main() {
    int x = 10;
    zmenHodnotu(&x); // Předáváme adresu proměnné x
    printf("Nove x: %d\n", x); // Vypíše 100
    return 0;
}
```

### Předávání polí jako argumentů
- V jazyce C se pole do funkcí **vždy předávají odkazem** (přesněji jako ukazatel na první prvek pole).
- To znamená, že pokud uvnitř funkce změníte prvek pole, **změní se i v původním poli**.
- Protože pole při předání do funkce "zdegeneruje" na ukazatel (tzv. *array decay*), **funkce nezná jeho velikost** (pomocí `sizeof` byste dostali jen velikost ukazatele). Velikost pole se proto obvykle musí předat jako další argument (výjimkou jsou textové řetězce ukončené `\0`).

```c
#include <stdio.h>

// Funkce přijímá pole a jeho velikost
// Zápis "int pole[]" je zde ekvivalentní k "int *pole"
void vynasobDvema(int pole[], int velikost) {
    for(int i = 0; i < velikost; i++) {
        pole[i] *= 2; // Upravuje přímo původní pole v paměti
    }
}

int main() {
    int cisla[] = {1, 2, 3, 4};
    int velikost = sizeof(cisla) / sizeof(cisla[0]);

    vynasobDvema(cisla, velikost); // Předáváme pole a jeho velikost

    for(int i = 0; i < velikost; i++) {
        printf("%d ", cisla[i]); // Vypíše: 2 4 6 8
    }
    return 0;
}
```

## 4. Lokální a globální proměnné
- **Lokální proměnné:** Deklarované uvnitř funkce. Existují pouze po dobu běhu této funkce a zvenčí k nim nelze přistoupit.
- **Globální proměnné:** Deklarované mimo všechny funkce (obvykle na začátku souboru). Jsou přístupné odevšad. Jejich používání se nedoporučuje, protože ztěžují sledování toku dat a mohou vést k chybám.

## 5. Rekurze
- Funkce volá sama sebe.
- Každá rekurzivní funkce musí mít **ukončovací podmínku** (tzv. base case), jinak dojde k nekonečnému zacyklení a přetečení zásobníku (Stack Overflow).
- Často se používá pro matematické operace jako výpočet faktoriálu nebo procházení stromových struktur.

```c
int faktorial(int n) {
    if (n <= 1) { // Ukončovací podmínka
        return 1;
    }
    return n * faktorial(n - 1); // Rekurzivní volání
}
```

## 6. Hlavní funkce `main()`
- Vstupní bod každého C programu. Operační systém při spuštění programu hledá právě funkci `main()` a začne ji vykonávat.
- Obvykle vrací `int` (`return 0;` znamená úspěšné ukončení programu, nenulová hodnota obvykle značí chybu).
