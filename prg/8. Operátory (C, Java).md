**Operátor** je symbol, který provádí určitou matematickou, relační nebo logickou operaci nad jedním nebo více **operandy** (hodnotami nebo proměnnými).

---

## 1. Aritmetické operátory
Slouží k základním matematickým výpočtům.
- `+` Sčítání
- `-` Odčítání
- `*` Násobení
- `/` Dělení (u celých čísel funguje jako celočíselné dělení: `5 / 2 = 2`)
- `%` Modulo (zbytek po celočíselném dělení: `5 % 2 = 1`)

## 2. Relační (porovnávací) operátory
Porovnávají dvě hodnoty. Výsledkem je v C celočíselná hodnota **1 (pravda - true)** nebo **0 (nepravda - false)**. V Javě je výsledkem typ `boolean` (`true`/`false`).
- `==` Rovná se (Pozor na velmi častou chybu - záměnu s přiřazením `=`)
- `!=` Nerovná se
- `>` Větší než
- `<` Menší než
- `>=` Větší nebo rovno
- `<=` Menší nebo rovno

## 3. Logické operátory
Používají se pro spojování podmínek.
- `&&` Logické AND (A zároveň) – vrací 1, pokud jsou obě podmínky pravdivé.
- `||` Logické OR (Nebo) – vrací 1, pokud je alespoň jedna podmínka pravdivá.
- `!` Logické NOT (Negace) – obrací pravdivostní hodnotu (z 1 udělá 0, z 0 udělá 1).

*Zkrácené vyhodnocování (Short-circuit evaluation):* Pokud je u `&&` první podmínka nepravda, druhá se už nevyhodnocuje. U `||` se druhá nevyhodnocuje, pokud je už první pravda.

## 4. Přiřazovací operátory
Slouží k uložení hodnoty do proměnné.
- `=` Základní přiřazení (`a = 5`)
- Zkrácené (složené) zápisy: `+=`, `-=`, `*=`, `/=`, `%=`
  - Příklad: `a += 3;` je to samé jako `a = a + 3;`

## 5. Inkrementace a Dekrementace
Speciální unární operátory pro zvýšení/snížení hodnoty o 1.
- `++` Inkrementace (zvýšení o 1)
- `--` Dekrementace (snížení o 1)

**Prefixový vs. Postfixový zápis:**
- **Prefix (`++a`)**: Hodnota se *nejprve* zvýší a až pak se použije ve výrazu.
- **Postfix (`a++`)**: Hodnota se *nejprve* použije ve výrazu a až pak se zvýší.
```c
int x = 5;
int y = ++x; // x se zvýší na 6, do y se uloží 6

int a = 5;
int b = a++; // do b se uloží 5, a se následně zvýší na 6
```

## 6. Bitové operátory
Pracují s daty na úrovni jednotlivých bitů (nul a jedniček). Jsou velmi důležité v C při low-level programování a sítích (např. maskování IP adres).
- `&` Bitové AND
- `|` Bitové OR
- `^` Bitové XOR (exclusive OR)
- `~` Bitové NOT (inverze bitů, unární)
- `<<` Posun doleva (Shift left) – efektivně násobí 2
- `>>` Posun doprava (Shift right) – efektivně dělí 2

## 7. Ternární operátor
Jediný operátor, který bere 3 operandy. Slouží jako zkrácený zápis pro jednoduché podmínky `if-else`.
- **Syntaxe:** `podmínka ? výraz_pokud_pravda : výraz_pokud_nepravda;`
```c
int vek = 18;
// Pokud je vek >= 18, uloží se 1 (true), jinak 0 (false)
int dospely = (vek >= 18) ? 1 : 0;
```

## 8. Operátor sizeof (C)
Speciální unární operátor v jazyce C, který vrací velikost datového typu nebo proměnné v bajtech. Je naprosto klíčový při dynamické alokaci paměti (např. pomocí `malloc`). V Javě tento operátor neexistuje (velikosti typů jsou tam pevně dané a paměť se spravuje automaticky).
- **Syntaxe:** `sizeof(typ)` nebo `sizeof(promenna)`
```c
int velikost_intu = sizeof(int); // Většinou vrátí 4 (bajty)
int pole[10];
int velikost_pole = sizeof(pole); // Vrátí 40 (10 prvků * 4 bajty)
```

## 9. Specifika operátorů v Javě
Ačkoliv Java sdílí většinu operátorů s jazykem C, existuje zde několik důležitých rozdílů a specifických operátorů:

- **Operátor `instanceof`:** Používá se k ověření, zda daný objekt je instancí určité třídy nebo rozhraní. Vrací `boolean` (`true`/`false`).
  ```java
  String text = "Ahoj";
  boolean jeString = text instanceof String; // true
  ```
- **Sčítání řetězců (String concatenation):** V Javě je operátor `+` přetížen pro spojování textových řetězců. Pokud je alespoň jeden operand typu `String`, Java automaticky převede druhý operand na řetězec a spojí je.
  ```java
  String vysledek = "Číslo je: " + 5; // "Číslo je: 5"
  ```
- **Nepříznakový bitový posun doprava (`>>>`):** Protože Java (na rozdíl od C) nemá nepříznakové (unsigned) datové typy (všechna celá čísla kromě `char` jsou se znaménkem), zavádí operátor `>>>`. Ten posune bity doprava a zleva doplňuje vždy nuly (zatímco klasický `>>` udržuje znaménkový bit).
- **Logické operátory bez zkráceného vyhodnocování (`&` a `|`):** V Javě lze jednoduché `&` a `|` použít i pro `boolean` hodnoty. Rozdíl oproti `&&` a `||` je ten, že se vyhodnotí **vždy obě strany** (nedochází k tzv. short-circuit vyhodnocování).
- **Absence ukazatelových operátorů:** V Javě neexistují ukazatele v pojetí jazyka C, proto zde nenajdeme operátory pro získání adresy (`&` jako unární operátor) ani pro dereferenci (`*`). Paměť a reference se spravují automaticky.

## Priorita a asociativita
- **Priorita** určuje, který operátor se vyhodnotí dříve (např. násobení `*` má vyšší prioritu než sčítání `+`).
- **Asociativita** určuje směr vyhodnocování, pokud mají operátory stejnou prioritu (většinou zleva doprava, ale např. přiřazení `=` jde zprava doleva).
- Pro změnu priority vždy používáme **kulaté závorky `()`**.
