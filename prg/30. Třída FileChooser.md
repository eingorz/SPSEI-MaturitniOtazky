
# Charakteristika

Třída JavaFX FileChooser slouží k zobrazení dialogového okna pro výběr souborů ze souborového systému. Umožňuje uživateli pohodlně vybrat soubor pro otevření nebo určit umístění pro uložení souboru.

Je součástí balíčku:

`javafx.stage.FileChooser`

Používá se hlavně v GUI aplikacích pro práci se soubory (např. textové editory, import/export dat apod.).

## Vlastnosti (Properties)
### 1. Title (titulek okna)
Nastavuje název dialogového okna
```Java
fileChooser.setTitle("Vyber soubor");
```
### 2. Initial Directory (počáteční složka)
Určuje výchozí adresář
```Java
fileChooser.setInitialDirectory(new File("C:/"));
```
### 3. Initial File Name (výchozí název souboru)
Používá se hlavně při ukládání
```Java
fileChooser.setInitialFileName("novy_soubor.txt");
```
### 4. Extension Filters (filtry přípon)
Omezuje typy zobrazovaných souborů
```Java
fileChooser.getExtensionFilters().add(
    new FileChooser.ExtensionFilter("Textové soubory", "*.txt")
);
```
### 5. Selected Extension Filter
Určuje aktuálně vybraný filtr (pokud není vybrán nebo není obsažen v seznamu, vybere se první filtr). Po zavření dialogu se hodnota této vlastnosti aktualizuje tak, aby odpovídala uživatelsky vybranému filtru rozšíření z dialogu.
```Java
// Vytvoření filteru
FileChooser.ExtensionFilter extFilter1 = FileChooser.ExtensionFilter("Textové soubory", "*.txt");
FileChooser.ExtensionFilter extFilter2 = FileChooser.ExtensionFilter("Všechny soubory", "*.*");

// přidání filterů
fileChooser.getExtensionFilters().addAll(extFilter1, extFilter2);

fileChooser.setSelectedExtensionFilter(extFilter1); // filtr který bude vybraný při spuštění
```
## Základní metody
### 1. showOpenDialog()
Otevře dialog pro výběr souboru
Vrací objekt `File`
```Java
File file = fileChooser.showOpenDialog(stage);
```
### 2. showSaveDialog()
Dialog pro uložení souboru
```Java
File file = fileChooser.showSaveDialog(stage);
```
### 3. showOpenMultipleDialog()
Umožňuje vybrat více souborů

```Java
List<File> files = fileChooser.showOpenMultipleDialog(stage);
```
Použití (praktický příklad)
```Java
FileChooser fileChooser = new FileChooser();
fileChooser.setTitle("Otevřít soubor");

fileChooser.getExtensionFilters().addAll(
    new FileChooser.ExtensionFilter("Textové soubory", "*.txt"),
    new FileChooser.ExtensionFilter("Všechny soubory", "*.*")
);

File file = fileChooser.showOpenDialog(stage);

if (file != null) {
    System.out.println("Vybraný soubor: " + file.getAbsolutePath());
}
```

### Typické využití
- Otevírání souborů (např. .txt, .csv)
- Ukládání dat do souboru
- Výběr obrázků
- Import/export dat

### Shrnutí
FileChooser je dialog pro práci se soubory v GUI aplikacích.
Umožňuje:
výběr souboru
uložení souboru
filtrování přípon
Je jednoduchý na použití a běžně používaný v aplikacích vytvořených pomocí JavaFX.
