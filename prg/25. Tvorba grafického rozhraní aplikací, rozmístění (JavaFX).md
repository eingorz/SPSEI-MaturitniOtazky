# 1. Soubory pro tvorbu JavaFX Application

JavaFX aplikace se obvykle skládá z několika typů souborů:

- Java třídy (.java)
    - Obsahují logiku aplikace
    - Hlavní třída dědí od Application
    - Přepisuje metodu start(Stage stage)
- FXML soubory (.fxml)
    - XML soubory pro definici grafického rozhraní
    - Oddělují vzhled od logiky
    - Používají se spolu s kontrolerem
- Controller (řadič)
    - Java třída propojená s FXML
    - Obsahuje obsluhu událostí (např. kliknutí na tlačítko)
- CSS soubory
    - Slouží ke stylování aplikace (barvy, fonty, velikosti)

# 2. Základní pojmy – Stage, Scene, Node
## Stage
Představuje okno aplikace
Je hlavní kontejner (např. hlavní okno programu)
V aplikaci může být více oken

```Java
stage.setTitle("Moje aplikace");
stage.setScene(scene); // Stage (okno) má scénu (obsah)
stage.show();
```

## Scene
- Obsahuje veškerý vizuální obsah
- Každý Stage má jednu Scene
- Má velikost
- Má kořenový komponent (root node)

```Java
Scene scene = new Scene(root, 400, 300);
stage.setScene(scene);
```

## Node
- Základní stavební prvek GUI
- Každý prvek (tlačítko, text, panel) je Node
- Všechny prvky tvoří hierarchickou stromovou strukturu

Příklady:
- Button
- Label
- TextField
- Pane

## Application

Třída Application je základ každé JavaFX aplikace.

Nachází se v balíčku javafx.application
Každá JavaFX aplikace od ní dědí
Slouží jako vstupní bod programu

Hlavní kroky:
- Spuštění pomocí launch(args)
- Automatické zavolání metody start(Stage stage)
- Při spuštění aplikace je metodě start zpřístupněna `Stage`
- V metodě start se vytváří GUI

```Java
public class Main extends Application {
    @Override
    public void start(Stage stage) {
        stage.setTitle("Moje aplikace");
        stage.show();
    }

    public static void main(String[] args) {
        launch(args);
    }
}
```

# 3. Rozložení (Layout)

Rozložení určuje, jak jsou prvky umístěny v okně.

## FXML soubory

- speciální XML formát používaný v JavaFX pro definici uživatelského rozhraní.
- Umožňuje oddělit vzhled (UI) od logiky (Java)
- UI se zapisuje deklarativně (bez programování v Javě)
- Používá se spolu s třídou FXMLLoader
- Může být propojen s controllerem pomocí fx:controller

Příklad FXML:

```XML
<?import javafx.scene.control.*?>
<?import javafx.scene.layout.VBox?>

<VBox spacing="10" alignment="CENTER"
      xmlns:fx="http://javafx.com/fxml"
      fx:controller="com.example.Controller">

    <Label text="Zadej jméno:" />

    <TextField fx:id="nameField" />

    <Button text="Potvrdit" onAction="#handleClick" />

    <Label fx:id="outputLabel" />

</VBox>
```

Načtení FXML:

```Java
FXMLLoader loader = new FXMLLoader(getClass().getResource("main.fxml"));
Parent root = loader.load();

Scene scene = new Scene(root);
stage.setScene(scene);
stage.show();
```

### Controller
- Obsahuje logiku pro prvky definované ve FXML
- Propojení přes fx:id a anotaci @FXML

```Java
@FXML
private Button myButton;

@FXML
private void handleClick() {
    System.out.println("Kliknuto!");
}
```

Nejčastější layouty:

## Pane
- Základní kontejner
- Prvky mají pevně dané souřadnice

## HBox
- Prvky vedle sebe (vodorovně)

```Java
HBox hbox = new HBox(10); // mezera 10 px
hbox.getChildren().addAll(btn1, btn2);
```

## VBox
- Prvky pod sebou (svisle)

## BorderPane
- Rozdělení na 5 částí:
    `top`, `bottom`, `left`, `right`, `center`

## GridPane
- Rozložení do tabulky (řádky a sloupce)

```Java
grid.add(button, 0, 0); // sloupec, řádek
```

## FlowPane
Prvky se automaticky zalamují (jako text)

# 4. Práce s komponentami

Komponenty (ovládací prvky) slouží k interakci s uživatelem.

Základní komponenty:
- Button – tlačítko
- Label – text
- TextField – vstupní pole
- CheckBox – zaškrtávací pole
- RadioButton – výběr jedné možnosti
- ComboBox – rozbalovací seznam
- Obsluha událostí

## Reakce na akce uživatele (např. kliknutí):

```Java
button.setOnAction(e -> {
    System.out.println("Kliknuto!");
});
```

## Práce s hodnotami

```Java
String text = textField.getText();
label.setText("Zadal jsi: " + text);
```

## Přidávání komponent do layoutu
```Java
VBox root = new VBox();
root.getChildren().addAll(label, textField, button);
```
