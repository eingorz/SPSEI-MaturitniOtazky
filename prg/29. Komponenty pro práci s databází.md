
# 1. Charakteristika a použití

V JavaFX se pro práci s databází kombinuje:
- JDBC (Java Database Connectivity) → komunikace s databází
- UI komponenty JavaFX → zobrazení a editace dat

Typický scénář:
- Připojení k databázi
- Načtení dat (SQL dotaz)
- Uložení do objektů (např. třída Person)
- Zobrazení v komponentě TableView

# 2. Připojení k databázi

Pro připojení se používá JDBC API.

Postup:
- Načtení driveru (dnes většinou automatické)
- Vytvoření spojení (Connection)
- Vytvoření dotazu (Statement / PreparedStatement)
- Získání výsledků (ResultSet)

Ukázka:
```Java
Connection conn = DriverManager.getConnection(
    "jdbc:mysql://localhost:3306/mojedb", "user", "heslo");

Statement stmt = conn.createStatement();
ResultSet rs = stmt.executeQuery("SELECT * FROM osoby");

while (rs.next()) {
    System.out.println(rs.getString("jmeno"));
}
```

Důležité:
používat PreparedStatement (bezpečnost – SQL injection)
zavírat spojení (close())

# 3. Komponenta TableView

TableView slouží k zobrazení dat v tabulce.

Základní vlastnosti:
items → seznam dat (ObservableList)
columns → sloupce tabulky
editable → možnost editace

Ukázka:
```Java
TableView<Person> table = new TableView<>();
ObservableList<Person> data = FXCollections.observableArrayList();

table.setItems(data);
```

Použití:
- zobrazení dat z databáze
- editace dat
- filtrování a třídění

Události:
- výběr řádku (selectedItemProperty)
- kliknutí (setOnMouseClicked)

# 4. Komponenta TableColumn

Reprezentuje jeden sloupec tabulky.

Základní vlastnosti:
- text → název sloupce
- cellValueFactory → odkud se bere hodnota
- cellFactory → jak se buňka vykreslí

Ukázka:
```Java
TableColumn<Person, String> colName = new TableColumn<>("Jméno");
colName.setCellValueFactory(
    new PropertyValueFactory<>("name"));
```

Poznámka:
název musí odpovídat atributu v třídě (getName())

# 5. Komponenta TableCell

Reprezentuje jednotlivou buňku.

Používá se při:
- vlastním vykreslení
- editaci buněk

Ukázka vlastní buňky:
```Java
colName.setCellFactory(column -> new TableCell<Person, String>() {
    @Override
    protected void updateItem(String item, boolean empty) {
        super.updateItem(item, empty);
        if (empty) {
            setText(null);
        } else {
            setText(item.toUpperCase());
        }
    }
});
```

# 6. Propojení databáze s TableView

Postup:
- Načtení dat z DB
- Uložení do objektů
- Přidání do ObservableList
- Nastavení do TableView

Ukázka:
```Java
ObservableList<Person> data = FXCollections.observableArrayList();

while (rs.next()) {
    data.add(new Person(
        rs.getString("jmeno"),
        rs.getInt("vek")
    ));
}

table.setItems(data);
```

# 7. Shrnutí
JDBC → připojení k databázi
TableView → zobrazení dat
TableColumn → definice sloupců
TableCell → vykreslení a editace buněk

Společně umožňují vytvořit GUI aplikaci pracující s databází (CRUD operace).
