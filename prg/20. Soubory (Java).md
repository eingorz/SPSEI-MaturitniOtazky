
## Soubory v jazyce Java

Java poskytuje **objektově orientovaný přístup k práci se soubory**.

Balíčky:
```Java
java.io  
java.nio
```

# Třída `File`

Reprezentuje soubor nebo adresář, který ovšem nemusí existovat.

```Java
File f = new File("data.txt");
```

Příklady metod:
- `exists()`
- `createNewFile()`
- `delete()`
- `length()`

# Práce s textovými soubory
## Čtení

## `FileReader`

Čtení textového souboru.
```Java
FileReader fr = new FileReader("data.txt");
```

## `BufferedReader`

Efektivní čtení po řádcích.
```Java
BufferedReader br = new BufferedReader(new FileReader("data.txt"));  
  
String line = br.readLine();
```

Příklad:
```Java
BufferedReader br = new BufferedReader(new FileReader("data.txt"));  
  
String line;  
while((line = br.readLine()) != null){  
    System.out.println(line);  
}  
  
br.close();
```

# Zápis do souboru

## `FileWriter`
```Java
FileWriter fw = new FileWriter("data.txt");
```

## `BufferedWriter`
```Java
BufferedWriter bw = new BufferedWriter(new FileWriter("data.txt"));  
  
bw.write("Hello");  
bw.newLine();  
  
bw.close();
```

# Práce s binárními soubory

## Proudy (streams)

Pro práci s binárními daty se používají:

## `InputStream` (čtení)

- čte data **po bajtech**
- základní metoda:

```Java
int read()
```

Vrací:
- hodnotu 0–255 (1 byte)
- `-1` = konec souboru 
## `OutputStream` (zápis)

- zapisuje data po bajtech

```Java
void write(int b)
```

## Nejdůležitější třídy

### `FileInputStream`

- čtení ze souboru

```Java
FileInputStream fis = new FileInputStream("data.bin");
```

### `FileOutputStream`

- zápis do souboru

```Java
FileOutputStream fos = new FileOutputStream("data.bin");
```

Pozor:
- přepíše soubor
- nebo lze přidávat:

```Java
new FileOutputStream("data.bin", true); // append
```

## Bufferované proudy

### Proč buffer?

Bez bufferu:
- čte se **1 bajt = pomalé**

S bufferem:
- čte se **blok dat = rychlé**

## `BufferedInputStream`

```Java
BufferedInputStream bis =  
    new BufferedInputStream(new FileInputStream("data.bin"));
```

## `BufferedOutputStream`

```Java
BufferedOutputStream bos =  
    new BufferedOutputStream(new FileOutputStream("data.bin"));
```
### **Příklad s bufferem**
```Java
BufferedInputStream in =  
    new BufferedInputStream(new FileInputStream("a.jpg"));  
BufferedOutputStream out =  
    new BufferedOutputStream(new FileOutputStream("b.jpg"));  
  
int b;  
while ((b = in.read()) != -1) {  
    out.write(b);  
}  
  
in.close();  
out.close();
```

Výrazně rychlejší než bez bufferu
## Práce s poli bajtů (efektivnější varianta)

Místo čtení po jednom bajtu:

```Java
byte[] buffer = new byte[1024];  
int pocet;  
  
while ((pocet = in.read(buffer)) != -1) {  
    out.write(buffer, 0, pocet);  
}
```

Nejlepší praxe (rychlé + efektivní)
## Datové proudy (`DataInputStream` / `DataOutputStream`)

Umožňují číst/zapisovat **konkrétní datové typy**:

- `int`
- `double`
- `boolean`
- `UTF String`
### **Zápis**

```Java
DataOutputStream dos =  
    new DataOutputStream(new FileOutputStream("data.bin"));  
  
dos.writeInt(100);  
dos.writeDouble(3.14);  
dos.writeBoolean(true);  
  
dos.close();
```
### **Čtení**

```Java
DataInputStream dis =  
    new DataInputStream(new FileInputStream("data.bin"));  
  
int i = dis.readInt();  
double d = dis.readDouble();  
boolean b = dis.readBoolean();  
  
dis.close();
```

*Poznámka: Číst musíme ve stejném pořadí, v jakém zapisujeme!*
## Serializace objektů

Umožňuje serializovat objekt do bytového proudu
### **Třídy**
- `ObjectOutputStream`
- `ObjectInputStream`

### **Příklad**

```Java
ObjectOutputStream oos =  
    new ObjectOutputStream(new FileOutputStream("obj.dat"));  
  
oos.writeObject(objekt);  
oos.close();

ObjectInputStream ois =  
    new ObjectInputStream(new FileInputStream("obj.dat"));  
  
Object obj = ois.readObject();  
ois.close();
```

Třída musí implementovat rozhraní `Serializable`

```Java
implements Serializable
```

# Uzavření čtecích/zápisových objektů

Po ukončení operace je vždy nutno objekt třídy `FileReader`/`BufferedReader` nebo `FileWriter`/`BufferedWriter` uzavřít metodou `close()` rozhraní `Closeable`.

Jelikož rozhraní `Closeable` využívá dědičnosti ke sdílení chování s rozhraním `AutoCloseable` je objekt tříd `FileReader`/`BufferedReader` nebo `FileWriter`/`BufferedWriter` schopen se sám uzavřít při destrukci nebo při jeho užití v `try` bloku takto:

```Java
try (FileWriter fw = new FileWriter("data.txt")) {
	// práce se souborem
} // fw se automaticky uzavře
```

# Moderní práce se soubory (Java NIO)

Balíček:
```Java
java.nio.file
```

Třídy:
- `Path`
- `Files`

Příklad čtení:
```Java
Path path = Path.of("data.txt");  
  
String content = Files.readString(path);
```

Zápis:
```Java
Files.writeString(path, "Hello world");
```

# Rozdíl mezi C a Java

|vlastnost|C|Java|
|---|---|---|
|přístup|procedurální|objektový|
|reprezentace souboru|`FILE*`|objekty (`File`, `Path`)|
|práce s pamětí|manuální|automatická (GC)|
|bezpečnost|nízká|vyšší|
|API|`stdio.h`|`java.io`, `java.nio`|
