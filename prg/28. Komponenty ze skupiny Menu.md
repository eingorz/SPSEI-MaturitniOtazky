# 1. Charakteristika a použití Menu komponent

Menu komponenty v JavaFX slouží k vytvoření klasické nabídky aplikace (menu), kterou uživatel zná z desktopových programů (např. „Soubor“, „Upravit“, „Nápověda“).
Používají se především pro:

- organizaci funkcí aplikace do přehledné struktury
- spouštění akcí (např. otevření souboru, uložení, ukončení)
- přepínání nastavení (např. zapnutí/vypnutí funkcí)

Menu je hierarchické – může obsahovat další položky i podmenu.

# 2. Základní vlastnosti a události
Vlastnosti:
`text` – název položky/menu
`graphic` – možnost přidat ikonu
`disable` – zakázání položky
`visible` – viditelnost
`accelerator` – klávesová zkratka (např. Ctrl+S)

Události:
`setOnAction()` – hlavní událost při kliknutí
`setOnShowing()` / setOnHiding() – při otevření/zavření menu
`selectedProperty()` – u přepínacích položek

# 3. Komponenta MenuBar (Controls)

MenuBar je hlavní kontejner pro menu.

umisťuje se typicky do horní části aplikace (např. do BorderPane → top)
obsahuje jednotlivá menu (Menu)
Příklad:
MenuBar menuBar = new MenuBar();

# 4. Komponenty Menu a MenuItem
## Menu
reprezentuje hlavní položku (např. „Soubor“)
může obsahovat další položky (MenuItem)

```Java
Menu fileMenu = new Menu("Soubor");
```

## MenuItem
konkrétní akce v menu (např. „Otevřít“, „Uložit“)
nelze dále větvit
```Java
MenuItem openItem = new MenuItem("Otevřít");
openItem.setOnAction(e -> System.out.println("Otevírám soubor"));
```

Přidání do menu:

```Java
fileMenu.getItems().add(openItem);
```

# 5. CheckMenuItem
položka s možností zaškrtnutí (ON/OFF)
používá se pro zapnutí/vypnutí funkcí
```Java
CheckMenuItem showToolbar = new CheckMenuItem("Zobrazit panel");
showToolbar.setSelected(true);

showToolbar.setOnAction(e -> {
    System.out.println("Stav: " + showToolbar.isSelected());
});
```

# 6. RadioMenuItem
umožňuje výběr pouze jedné možnosti ze skupiny
používá se společně s ToggleGroup
```Java
RadioMenuItem light = new RadioMenuItem("Světlý režim");
RadioMenuItem dark = new RadioMenuItem("Tmavý režim");

ToggleGroup group = new ToggleGroup();
light.setToggleGroup(group);
dark.setToggleGroup(group);
```

# 7. Separator (SeparatorMenuItem)
slouží k vizuálnímu oddělení položek v menu
zlepšuje přehlednost

```Java
SeparatorMenuItem separator = new SeparatorMenuItem();
```

# 8. Kompletní ukázka
```JAva
MenuBar menuBar = new MenuBar();

Menu fileMenu = new Menu("Soubor");
MenuItem open = new MenuItem("Otevřít");
MenuItem exit = new MenuItem("Konec");

exit.setOnAction(e -> System.exit(0));

fileMenu.getItems().addAll(open, new SeparatorMenuItem(), exit);

Menu viewMenu = new Menu("Zobrazení");
CheckMenuItem toolbar = new CheckMenuItem("Panel nástrojů");

Menu themeMenu = new Menu("Motiv");
RadioMenuItem light = new RadioMenuItem("Světlý");
RadioMenuItem dark = new RadioMenuItem("Tmavý");

ToggleGroup tg = new ToggleGroup();
light.setToggleGroup(tg);
dark.setToggleGroup(tg);

themeMenu.getItems().addAll(light, dark);
viewMenu.getItems().addAll(toolbar, themeMenu);

menuBar.getMenus().addAll(fileMenu, viewMenu);
```

# 9. Shrnutí
MenuBar = hlavní panel menu
Menu = skupina položek
MenuItem = konkrétní akce
CheckMenuItem = přepínač (ano/ne)
RadioMenuItem = výběr jedné z možností
SeparatorMenuItem = oddělení položek
hlavní událost: setOnAction()
