# SPŠEI - Maturitní otázky
- Tento repozitář je určen k přežití maturitní sezóny 2025/2026 na Střední Průmyslové Škole Čar a Kouzel v Ostravě.
- Obsahuje otázky z předmětů:
  - WEA - Webové Aplikace
  - PRG - Programování
  - DAS - Databázové Systémy
  - ČJL - Český Jazyk a Literatura (Díla z povinné četby)
